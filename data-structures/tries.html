Tries (Prefix Trees)

A tree where each edge is a character, so every node represents a prefix. Words that share a prefix share a path — giving O(length) insert and search, independent of how many words you've stored.

The Shape

   Insert: "cat", "car", "card", "dog"

         (root)
         /    \
        c      d
        |      |
        a      o
       / \     |
      t   r    g●
      ●   |●
          d●          ● = end of a word

   "cat" and "car" share the "ca" path → no duplication.

Code: Trie (insert / search / prefix)

struct Trie {
    struct Node { Node* next[26] = {}; bool end = false; };
    Node* root = new Node();

    void insert(const string& s) {
        Node* cur = root;
        for (char c : s) {
            int i = c - 'a';
            if (!cur->next[i]) cur->next[i] = new Node();
            cur = cur->next[i];
        }
        cur->end = true;
    }
    bool search(const string& s) {
        Node* cur = walk(s);
        return cur && cur->end;
    }
    bool startsWith(const string& p) { return walk(p) != nullptr; }
  private:
    Node* walk(const string& s) {
        Node* cur = root;
        for (char c : s) {
            cur = cur->next[c - 'a'];
            if (!cur) return nullptr;
        }
        return cur;
    }
};
class Trie {
    static class Node { Node[] next = new Node[26]; boolean end; }
    Node root = new Node();

    void insert(String s) {
        Node cur = root;
        for (char c : s.toCharArray()) {
            int i = c - 'a';
            if (cur.next[i] == null) cur.next[i] = new Node();
            cur = cur.next[i];
        }
        cur.end = true;
    }
    boolean search(String s) { Node c = walk(s); return c != null && c.end; }
    boolean startsWith(String p) { return walk(p) != null; }
    private Node walk(String s) {
        Node cur = root;
        for (char c : s.toCharArray()) {
            cur = cur.next[c - 'a'];
            if (cur == null) return null;
        }
        return cur;
    }
}
class Trie:
    def __init__(self):
        self.children = {}
        self.end = False

    def insert(self, word):
        node = self
        for ch in word:
            if ch not in node.children:
                node.children[ch] = Trie()
            node = node.children[ch]
        node.end = True

    def _walk(self, s):
        node = self
        for ch in s:
            if ch not in node.children:
                return None
            node = node.children[ch]
        return node

    def search(self, word):
        node = self._walk(word)
        return node is not None and node.end

    def starts_with(self, prefix):
        return self._walk(prefix) is not None

The XOR Trie (a CP favorite)

  • Store numbers as their binary bits in a trie (one bit per level, MSB first).
  • To find the max XOR with a query, greedily walk toward the opposite bit at each level.
  • Solves "maximum XOR pair / subarray" in O(n · bits) — a classic interview & contest problem.

What Tries Power

Autocomplete / prefix search — all words under a prefix node.

Spell-check & dictionaries — fast word membership.

Maximum XOR — binary (bitwise) trie.

Word games / Boggle — prune dead prefixes early.

IP routing — longest-prefix match on bits.

Common Pitfalls

1. Memory: a 26-pointer array per node is wasteful for sparse data — use a hash map of children.

2. End flag vs prefix: "car" being stored doesn't mean "ca" is a word — track end separately from existence.

3. Deletion: only prune nodes with no children and not marking another word's end.

4. Alphabet size: map characters to indices consistently (lowercase, digits, etc.).

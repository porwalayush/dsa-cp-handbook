Dynamic Segment Tree

A Dynamic Segment Tree (also called an implicit or sparse segment tree) creates nodes only when they are actually needed. This lets you handle ranges as large as [0, 10^9] while using memory proportional to the number of operations, not the range size.

When to Use

  • The value range is huge (e.g., [0, 10^9] or [0, 10^18])
  • But the number of actual updates/queries is small (e.g., 10^5)
  • You cannot (or don't want to) apply coordinate compression
  • Queries arrive online and you cannot pre-process all positions
Rule of thumb: If range N is huge but operations Q are few, a dynamic segment tree uses O(Q log N) memory instead of O(N).

Concept: Create Nodes on Demand

In a standard segment tree over [0, N-1], we allocate 4N entries. When N = 10^9, that is 16 GB of memory for integers alone. A dynamic segment tree starts with just a root and creates child nodes only when an update touches that region.

Each node stores pointers (or indices) to its left and right children instead of using the 2*i / 2*i+1 formula. If a child has never been accessed, its pointer remains null (or 0), implicitly representing a subtree of all zeros.

When to prefer coordinate compression:
  • All queries are known offline (competitive programming with offline approach)
  • Simpler code is a priority and memory is tight
When to prefer dynamic segment tree:
  • Online queries (cannot see future queries)
  • Need to merge segment trees (e.g., small-to-large merge on trees)
  • Persistent segment tree required

Implementation: Node Pool for Cache Efficiency

Instead of using new/heap allocation per node (which causes cache misses), we pre-allocate a node pool (a flat array of node structs). Each "pointer" is just an index into this pool. This gives us:

  • Cache locality - nodes are contiguous in memory
  • No memory fragmentation - predictable allocation
  • Fast allocation - just increment a counter
  • Easy reset - set counter back to 0 to "free" everything
Node Pool (array-based):
+-------+-------+-------+-------+-------+-------+-------+
| idx 0 | idx 1 | idx 2 | idx 3 | idx 4 | idx 5 | idx 6 | ...
+-------+-------+-------+-------+-------+-------+-------+
| root  | L of 0| R of 0| L of 1| R of 1| L of 2|       |
| sum=15| sum=5 | sum=10| sum=5 | (null)| (null)| sum=10|
| L=1   | L=3   | L=0   | L=0   | ...   |       | L=0   |
| R=2   | R=4   | R=6   | R=5   |       |       | R=....|
+-------+-------+-------+-------+-------+-------+-------+
           ^                                        ^
     pool_cnt increments as nodes are created

Each node: { sum, left_child_idx, right_child_idx }
Null pointer = 0 (index 0 is reserved or sum=0 sentinel)

Code: Pointer-Based Version

#include <bits/stdc++.h>
using namespace std;

struct Node {
    long long sum;
    Node *left, *right;
    Node() : sum(0), left(nullptr), right(nullptr) {}
};

class DynamicSegTree {
    Node* root;
    int L, R;  // range [L, R]

public:
    DynamicSegTree(int l, int r) : L(l), R(r) {
        root = new Node();
    }

    void update(int pos, long long val) {
        update(root, L, R, pos, val);
    }

    long long query(int ql, int qr) {
        return query(root, L, R, ql, qr);
    }

private:
    void update(Node* node, int l, int r, int pos, long long val) {
        if (l == r) {
            node->sum += val;
            return;
        }
        int mid = l + (r - l) / 2;
        if (pos <= mid) {
            if (!node->left) node->left = new Node();
            update(node->left, l, mid, pos, val);
        } else {
            if (!node->right) node->right = new Node();
            update(node->right, mid + 1, r, pos, val);
        }
        long long ls = node->left ? node->left->sum : 0;
        long long rs = node->right ? node->right->sum : 0;
        node->sum = ls + rs;
    }

    long long query(Node* node, int l, int r, int ql, int qr) {
        if (!node || qr < l || r < ql) return 0;
        if (ql <= l && r <= qr) return node->sum;
        int mid = l + (r - l) / 2;
        return query(node->left, l, mid, ql, qr) +
               query(node->right, mid + 1, r, ql, qr);
    }
};

int main() {
    DynamicSegTree dst(0, 1000000000);
    dst.update(5, 10);
    dst.update(999999999, 7);
    dst.update(42, 3);
    cout << dst.query(0, 1000000000) << "\n";  // 20
    cout << dst.query(5, 42) << "\n";           // 13
}

    static class Node {
        long sum;
        Node left, right;
        Node() { sum = 0; left = null; right = null; }
    }

    Node root;
    int L, R;

    DynamicSegTree(int l, int r) {
        L = l; R = r;
        root = new Node();
    }

    void update(int pos, long val) {
        update(root, L, R, pos, val);
    }

    long query(int ql, int qr) {
        return query(root, L, R, ql, qr);
    }

    private void update(Node node, int l, int r, int pos, long val) {
        if (l == r) {
            node.sum += val;
            return;
        }
        int mid = l + (r - l) / 2;
        if (pos <= mid) {
            if (node.left == null) node.left = new Node();
            update(node.left, l, mid, pos, val);
        } else {
            if (node.right == null) node.right = new Node();
            update(node.right, mid + 1, r, pos, val);
        }
        long ls = node.left != null ? node.left.sum : 0;
        long rs = node.right != null ? node.right.sum : 0;
        node.sum = ls + rs;
    }

    private long query(Node node, int l, int r, int ql, int qr) {
        if (node == null || qr < l || r < ql) return 0;
        if (ql <= l && r <= qr) return node.sum;
        int mid = l + (r - l) / 2;
        return query(node.left, l, mid, ql, qr) +

    }

    }
}

class Node:
    __slots__ = ['sum', 'left', 'right']
    def __init__(self):
        self.sum = 0
        self.left = None
        self.right = None

class DynamicSegTree:
    def __init__(self, l, r):
        self.L, self.R = l, r
        self.root = Node()

    def update(self, pos, val):
        self._update(self.root, self.L, self.R, pos, val)

    def query(self, ql, qr):
        return self._query(self.root, self.L, self.R, ql, qr)

    def _update(self, node, l, r, pos, val):
        if l == r:
            node.sum += val
            return
        mid = (l + r) // 2
        if pos <= mid:
            if not node.left:
                node.left = Node()
            self._update(node.left, l, mid, pos, val)
        else:
            if not node.right:
                node.right = Node()
            self._update(node.right, mid + 1, r, pos, val)
        ls = node.left.sum if node.left else 0
        rs = node.right.sum if node.right else 0
        node.sum = ls + rs

    def _query(self, node, l, r, ql, qr):
        if not node or qr < l or r < ql:
            return 0
        if ql <= l and r <= qr:
            return node.sum
        mid = (l + r) // 2
        return (self._query(node.left, l, mid, ql, qr) +
                self._query(node.right, mid + 1, r, ql, qr))

Code: Array-Pool Version (Cache-Friendly)

#include <bits/stdc++.h>
using namespace std;

const int MAXNODES = 4000000;  // Q * log(N) upper bound

struct Node {
    long long sum;
    int left, right;  // indices into pool (0 = null)
} pool[MAXNODES];

int pool_cnt = 1;  // 0 is reserved as null sentinel

int newNode() {
    pool[pool_cnt] = {0, 0, 0};
    return pool_cnt++;
}

void update(int node, int l, int r, int pos, long long val) {
    if (l == r) {
        pool[node].sum += val;
        return;
    }
    int mid = l + (r - l) / 2;
    if (pos <= mid) {
        if (!pool[node].left) pool[node].left = newNode();
        update(pool[node].left, l, mid, pos, val);
    } else {
        if (!pool[node].right) pool[node].right = newNode();
        update(pool[node].right, mid + 1, r, pos, val);
    }
    pool[node].sum = pool[pool[node].left].sum + pool[pool[node].right].sum;
}

long long query(int node, int l, int r, int ql, int qr) {
    if (!node || qr < l || r < ql) return 0;
    if (ql <= l && r <= qr) return pool[node].sum;
    int mid = l + (r - l) / 2;
    return query(pool[node].left, l, mid, ql, qr) +
           query(pool[node].right, mid + 1, r, ql, qr);
}

int main() {
    const int N = 1000000000;
    int root = newNode();  // root is pool index 1

    update(root, 0, N, 5, 10);
    update(root, 0, N, 999999999, 7);
    update(root, 0, N, 42, 3);

    cout << query(root, 0, N, 0, N) << "\n";   // 20
    cout << query(root, 0, N, 5, 42) << "\n";  // 13
    cout << "Nodes used: " << pool_cnt << "\n";
}

class DynamicSegTreePool {
    static final int MAXNODES = 4000000;
    static long[] sum = new long[MAXNODES];
    static int[] left = new int[MAXNODES];
    static int[] right = new int[MAXNODES];
    static int poolCnt = 1;  // 0 = null sentinel

    static int newNode() {
        sum[poolCnt] = 0;
        left[poolCnt] = 0;
        right[poolCnt] = 0;
        return poolCnt++;
    }

    static void update(int node, int l, int r, int pos, long val) {
        if (l == r) {
            sum[node] += val;
            return;
        }
        int mid = l + (r - l) / 2;
        if (pos <= mid) {
            if (left[node] == 0) left[node] = newNode();
            update(left[node], l, mid, pos, val);
        } else {
            if (right[node] == 0) right[node] = newNode();
            update(right[node], mid + 1, r, pos, val);
        }
        sum[node] = sum[left[node]] + sum[right[node]];
    }

    static long query(int node, int l, int r, int ql, int qr) {
        if (node == 0 || qr < l || r < ql) return 0;
        if (ql <= l && r <= qr) return sum[node];
        int mid = l + (r - l) / 2;
        return query(left[node], l, mid, ql, qr) +
               query(right[node], mid + 1, r, ql, qr);
    }

    public static void main(String[] args) {
        int N = 1000000000;
        int root = newNode();

        update(root, 0, N, 5, 10);
        update(root, 0, N, 999999999, 7);
        update(root, 0, N, 42, 3);

        System.out.println(query(root, 0, N, 0, N));   // 20
        System.out.println(query(root, 0, N, 5, 42));  // 13
        System.out.println("Nodes used: " + poolCnt);
    }
}

class DynamicSegTreePool:
    def __init__(self, lo, hi, max_nodes=4000000):
        self.lo, self.hi = lo, hi
        self.sum = [0] * max_nodes
        self.left = [0] * max_nodes   # 0 = null
        self.right = [0] * max_nodes
        self.cnt = 1  # reserve 0 as null
        self.root = self._new_node()

    def _new_node(self):
        idx = self.cnt
        self.cnt += 1
        return idx

    def update(self, pos, val):
        self._update(self.root, self.lo, self.hi, pos, val)

    def query(self, ql, qr):
        return self._query(self.root, self.lo, self.hi, ql, qr)

    def _update(self, node, l, r, pos, val):
        if l == r:
            self.sum[node] += val
            return
        mid = (l + r) // 2
        if pos <= mid:
            if not self.left[node]:
                self.left[node] = self._new_node()
            self._update(self.left[node], l, mid, pos, val)
        else:
            if not self.right[node]:
                self.right[node] = self._new_node()
            self._update(self.right[node], mid + 1, r, pos, val)
        self.sum[node] = self.sum[self.left[node]] + self.sum[self.right[node]]

    def _query(self, node, l, r, ql, qr):
        if not node or qr < l or r < ql:
            return 0
        if ql <= l and r <= qr:
            return self.sum[node]
        mid = (l + r) // 2
        return (self._query(self.left[node], l, mid, ql, qr) +
                self._query(self.right[node], mid + 1, r, ql, qr))

# Usage
dst = DynamicSegTreePool(0, 10**9)
dst.update(5, 10)
dst.update(999999999, 7)
dst.update(42, 3)
print(dst.query(0, 10**9))  # 20
print(dst.query(5, 42))      # 13
print(f"Nodes used: {dst.cnt}")

Memory Analysis

Each update creates at most O(log N) new nodes (one per level of the tree). With Q operations over range [0, N-1]:

ParameterValueNodes Created
N = 10^9, Q = 10^5log2(10^9) ~ 30At most 3 * 10^6 nodes
N = 10^18, Q = 10^5log2(10^18) ~ 60At most 6 * 10^6 nodes
N = 10^9, Q = 10^6log2(10^9) ~ 30At most 30 * 10^6 nodes
Memory formula: O(Q * log N) nodes total. Each node stores a value + 2 child pointers. In the pool version with long long sum + 2 int children, that is 16 bytes per node. For Q=10^5, N=10^9: about 48 MB worst case.

Applications

1. Online Queries (Cannot Compress)

When queries and updates come one at a time and you cannot see the future, coordinate compression is impossible. Dynamic segment trees handle this naturally.

2. Interval Scheduling / Sweep Line

Problems where intervals span [0, 10^9] but only ~10^5 intervals exist. Track coverage counts or maximum overlap without discretizing the full range.

3. Persistent Segment Tree Foundation

Persistent segment trees are built on the same "create nodes as needed" principle. Each version shares most nodes with previous versions, creating only O(log N) new nodes per update.

4. Merge-Based Problems (Small-to-Large)

When merging segment trees in DSU-on-tree or heavy-light problems, dynamic trees let you merge only the nodes that actually exist, avoiding O(N) per merge.

5. 2D Problems with Sparse Second Dimension

A segment tree of segment trees where the inner trees are dynamic: the outer tree covers rows [0, N], and each inner tree covers columns [0, M] but only allocates nodes for columns that have been updated in that row range.

Key Takeaways:
  • Use a dynamic segment tree when N is large but Q is small
  • Prefer the array-pool version in competitive programming for speed
  • Pointer-based version is cleaner for production code or when memory pooling is complex
  • Always estimate memory: Q * log(N) * (bytes per node)
  • Consider coordinate compression first if all queries are offline

How to Identify DP Problems

Master these patterns to recognize when DP applies

1

Optimal Substructure

Optimal solution uses optimal solutions of subproblems. "Find minimum/maximum..."

2

Overlapping Subproblems

Same subproblems solved repeatedly. Recursion tree has repeated nodes.

3

State Definition

What info needed to describe current situation? dp[i] = best answer considering first i items.

4

Transition Formula

How to get dp[i] from previous states? dp[i] = max(dp[i-1], dp[i-2] + val[i]).

5

Base Cases

Smallest subproblems with known answers. dp[0] = 0, dp[1] = 1. Starting conditions.

6

Problem Signals

"Count ways", "minimum cost", "maximum value", "is it possible", "longest/shortest".

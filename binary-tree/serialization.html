Serialize & Deserialize a Binary Tree

Serialization is the process of converting a tree (an in-memory pointer structure) into a flat, linear representation — usually a string or byte stream — so it can be stored on disk or sent over a network. Deserialization is the exact inverse: rebuilding the original tree from that flat representation.

The two operations are packaged together as a Codec: one method serialize(root) produces a string, and a matching method deserialize(data) consumes that same string to reconstruct an identical tree. The only hard requirement is the round-trip property: deserialize(serialize(root)) must yield a tree structurally equal to root.

Why Null Markers Are Essential

A plain traversal of a binary tree is ambiguous. Consider just a preorder list of values like 1, 2, 3. Many different trees produce that exact sequence — there is no way to know whether 2 is a left child, whether 3 hangs off 1 or off 2, and so on. One traversal alone does not encode shape, only order.

The fix is to explicitly record the absence of a child with a null marker (we use #). Once every missing child is written down, a single traversal becomes a bijection — exactly one tree maps to exactly one string and vice versa. This is what makes a single preorder (or postorder, or level-order) walk sufficient to rebuild the tree.

Without null markers, "1 2 3" is ambiguous:

    1            1             1
   /            / \             \
  2            2   3             2
 /                                \
3                                  3

With null markers, each shape has a UNIQUE string.

Preorder With Null Markers

The most common contest/interview format. We walk the tree in preorder (root, then left subtree, then right subtree). At each step we emit the node's value, and when we hit an empty pointer we emit the null marker #. Values are joined by a delimiter (a comma) so multi-digit and negative numbers survive intact.

Tree:                 Preorder walk (root, left, right):

      1               visit 1
     / \              visit 2
    2   3             2.left  = null  -> #
       / \            2.right = null  -> #
      4   5           visit 3
                      visit 4
                      4.left  = null  -> #
                      4.right = null  -> #
                      visit 5
                      5.left  = null  -> #
                      5.right = null  -> #

Serialized string:  "1,2,#,#,3,4,#,#,5,#,#"

Deserialize reads the tokens left to right through a shared cursor (a queue or an index/iterator). Because the tokens were emitted in preorder, we rebuild in the same order: read one token; if it is # return null; otherwise create the node, then recursively build its left subtree, then its right subtree. The cursor advances exactly once per token, so the recursion consumes the stream in lockstep with how it was produced — guaranteeing deserialize inverts serialize.

struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Codec {
public:
    // Preorder: root, left, right. '#' marks a null child.
    string serialize(TreeNode* root) {
        string out;
        encode(root, out);
        return out;
    }

    TreeNode* deserialize(string data) {
        stringstream ss(data);
        return decode(ss);
    }

private:
    void encode(TreeNode* node, string& out) {
        if (!node) { out += "#,"; return; }
        out += to_string(node->val) + ",";
        encode(node->left, out);
        encode(node->right, out);
    }

    TreeNode* decode(stringstream& ss) {
        string token;
        getline(ss, token, ',');          // read next token
        if (token == "#") return nullptr;
        TreeNode* node = new TreeNode(stoi(token));
        node->left  = decode(ss);            // build left first (preorder)
        node->right = decode(ss);            // then right
        return node;
    }
};

class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int x) { val = x; }
}

public class Codec {
    // Preorder: root, left, right. '#' marks a null child.
    public String serialize(TreeNode root) {
        StringBuilder sb = new StringBuilder();
        encode(root, sb);
        return sb.toString();
    }

    public TreeNode deserialize(String data) {
        Deque<String> tokens =
            new ArrayDeque<>(Arrays.asList(data.split(",")));
        return decode(tokens);
    }

    private void encode(TreeNode node, StringBuilder sb) {
        if (node == null) { sb.append("#,"); return; }
        sb.append(node.val).append(",");
        encode(node.left, sb);
        encode(node.right, sb);
    }

    private TreeNode decode(Deque<String> tokens) {
        String token = tokens.poll();        // read next token
        if (token.equals("#")) return null;
        TreeNode node = new TreeNode(Integer.parseInt(token));
        node.left  = decode(tokens);        // build left first (preorder)
        node.right = decode(tokens);        // then right
        return node;
    }
}

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val, self.left, self.right = val, left, right

class Codec:
    # Preorder: root, left, right. '#' marks a null child.
    def serialize(self, root):
        out = []
        def encode(node):
            if not node:
                out.append("#")
                return
            out.append(str(node.val))
            encode(node.left)
            encode(node.right)
        encode(root)
        return ",".join(out)

    def deserialize(self, data):
        tokens = iter(data.split(","))
        def decode():
            token = next(tokens)            # read next token
            if token == "#":
                return None
            node = TreeNode(int(token))
            node.left  = decode()           # build left first (preorder)
            node.right = decode()           # then right
            return node
        return decode()

Level-Order (BFS) Serialization

This is the format LeetCode uses to display trees: [1,2,3,null,null,4,5]. We perform a breadth-first traversal with a queue, emitting each node's value (or null for an empty child) level by level.

Tree:                 BFS order (level by level):

      1               level 0:  1
     / \              level 1:  2  3
    2   3             level 2:  (2's kids) null null
       / \                      (3's kids) 4    5
      4   5

LeetCode form:  [1,2,3,null,null,4,5]

Note: trailing nulls for the deepest leaves (4,5) are
trimmed by LeetCode's display, but emitting them is fine.

Serialize: push the root, then repeatedly pop a node — if it is non-null, emit its value and enqueue both children (even null ones); if it is null, emit # and enqueue nothing. Deserialize: read the first token as the root and enqueue it. Then for each dequeued parent, consume the next two tokens as its left and right children, attaching and enqueuing any non-null child. The queue links parents to children in the same order BFS visited them, so the rebuild is exact.

class Codec {
public:
    // Level-order (BFS). '#' marks a null slot.
    string serialize(TreeNode* root) {
        if (!root) return "";
        string out;
        queue<TreeNode*> q;
        q.push(root);
        while (!q.empty()) {
            TreeNode* node = q.front(); q.pop();
            if (!node) { out += "#,"; continue; }
            out += to_string(node->val) + ",";
            q.push(node->left);             // enqueue both children
            q.push(node->right);
        }
        return out;
    }

    TreeNode* deserialize(string data) {
        if (data.empty()) return nullptr;
        stringstream ss(data);
        string token;
        getline(ss, token, ',');
        TreeNode* root = new TreeNode(stoi(token));
        queue<TreeNode*> q;
        q.push(root);
        while (!q.empty()) {
            TreeNode* node = q.front(); q.pop();
            if (getline(ss, token, ',') && token != "#") {
                node->left = new TreeNode(stoi(token));
                q.push(node->left);
            }
            if (getline(ss, token, ',') && token != "#") {
                node->right = new TreeNode(stoi(token));
                q.push(node->right);
            }
        }
        return root;
    }
};

public class Codec {
    // Level-order (BFS). '#' marks a null slot.
    public String serialize(TreeNode root) {
        if (root == null) return "";
        StringBuilder sb = new StringBuilder();
        Queue<TreeNode> q = new LinkedList<>();
        q.add(root);
        while (!q.isEmpty()) {
            TreeNode node = q.poll();
            if (node == null) { sb.append("#,"); continue; }
            sb.append(node.val).append(",");
            q.add(node.left);              // enqueue both children
            q.add(node.right);
        }
        return sb.toString();
    }

    public TreeNode deserialize(String data) {
        if (data.isEmpty()) return null;
        String[] t = data.split(",");
        TreeNode root = new TreeNode(Integer.parseInt(t[0]));
        Queue<TreeNode> q = new LinkedList<>();
        q.add(root);
        int i = 1;
        while (!q.isEmpty()) {
            TreeNode node = q.poll();
            if (!t[i].equals("#")) {
                node.left = new TreeNode(Integer.parseInt(t[i]));
                q.add(node.left);
            }
            i++;
            if (!t[i].equals("#")) {
                node.right = new TreeNode(Integer.parseInt(t[i]));
                q.add(node.right);
            }
            i++;
        }
        return root;
    }
}

from collections import deque

class Codec:
    # Level-order (BFS). '#' marks a null slot.
    def serialize(self, root):
        if not root:
            return ""
        out, q = [], deque([root])
        while q:
            node = q.popleft()
            if not node:
                out.append("#")
                continue
            out.append(str(node.val))
            q.append(node.left)            # enqueue both children
            q.append(node.right)
        return ",".join(out)

    def deserialize(self, data):
        if not data:
            return None
        t = data.split(",")
        root = TreeNode(int(t[0]))
        q = deque([root])
        i = 1
        while q:
            node = q.popleft()
            if t[i] != "#":
                node.left = TreeNode(int(t[i]))
                q.append(node.left)
            i += 1
            if t[i] != "#":
                node.right = TreeNode(int(t[i]))
                q.append(node.right)
            i += 1
        return root
Preorder vs Level-order — which to pick?
  • Preorder (DFS): shortest code, naturally recursive, easy to reason about. Best for interviews and most contest needs.
  • Level-order (BFS): matches LeetCode's display format, iterative (no deep recursion / stack-overflow risk on skewed trees), and human-readable as a "tree by levels".
  • Postorder also works: serialize as left, right, root. To deserialize you read tokens from the back and build right before left (mirror of preorder), because the last value written is the root and the right subtree was the last child emitted.

Special Case: Serializing a BST

For a Binary Search Tree you can drop the null markers entirely. A bare preorder list such as 5,3,2,4,8,7,9 is already unambiguous: the BST ordering invariant plus value bounds tell you exactly where each node belongs. During deserialization you carry a (lower, upper) bound; a node attaches as a left or right child only if its value fits the bound, which uniquely reconstructs the shape.

# BST: preorder WITHOUT null markers is enough.
def serialize(root):
    out = []
    def pre(n):
        if n:
            out.append(str(n.val)); pre(n.left); pre(n.right)
    pre(root)
    return ",".join(out)

def deserialize(data):
    if not data: return None
    vals = list(map(int, data.split(",")))
    self_i = [0]
    def build(lower, upper):
        if self_i[0] == len(vals): return None
        v = vals[self_i[0]]
        if v < lower or v > upper: return None  # out of bounds
        self_i[0] += 1
        node = TreeNode(v)
        node.left  = build(lower, v)       # left values < v
        node.right = build(v, upper)       # right values > v
        return node
    return build(float("-inf"), float("inf"))

Edge Cases

  • Empty tree: serialize(null) must produce a string that deserializes back to null. Preorder yields "#"; the BFS version above returns "" and special-cases it on the way back.
  • Single node: a lone root with no children serializes (preorder) to "1,#,#" and must rebuild to a node with both children null.
  • Negative and multi-digit values: never serialize one character per node. A value like -12 or 305 needs a real delimiter (a comma) between tokens — otherwise -12 and 1,2 become indistinguishable. Always split on the delimiter, never on character position.
  • Duplicate values: fine for a general binary tree (shape is encoded explicitly), but the BST-without-markers trick assumes a strict ordering — decide up front whether duplicates go left or right and keep the bound comparison consistent.
Common bug: using a single character per node (e.g. "12##34####"). This silently breaks the moment a value is negative or has more than one digit. Use a comma-delimited token stream and parse whole tokens.

Complexity Analysis

OperationTimeSpace
Serialize (preorder)O(n)O(n)
Deserialize (preorder)O(n)O(n)
Serialize (level-order)O(n)O(n)
Deserialize (level-order)O(n)O(n)

Every node and every null marker is visited exactly once in each direction, so all four operations are O(n) time. The output string holds O(n) tokens, and the recursion stack or BFS queue holds at most O(n) entries, so space is O(n) as well.

Key insight: a raw traversal records order but not shape, so it maps many trees to one string — it is not invertible. Adding an explicit null marker for every missing child restores the shape information, turning an ambiguous traversal into a bijection between trees and strings. That single change is what makes deserialize a true inverse of serialize.

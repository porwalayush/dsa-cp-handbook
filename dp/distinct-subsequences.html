Distinct Subsequences & Word Break

Two classic string DP problems: counting subsequences that match a target, and determining if a string can be segmented into dictionary words.

Problem 1: Distinct Subsequences

Given strings s and t, count the number of distinct subsequences of s that equal t. A subsequence is formed by deleting some (or no) characters from s without changing the relative order of the remaining characters.

Example: s = "rabbbit", t = "rabbit"
Answer: 3

The 3 ways (marking which 'b' we skip):
  r a b b [b] i t  (skip 3rd b)
  r a b [b] b i t  (skip 2nd b)
  r a [b] b b i t  (skip 1st b)

Each gives the subsequence "rabbit"

State Definition

State: dp[i][j] = number of distinct subsequences of s[0..i-1] that equal t[0..j-1]

Transition:
If s[i-1] == t[j-1]: dp[i][j] = dp[i-1][j-1] + dp[i-1][j]
  (use s[i-1] to match t[j-1], OR skip s[i-1])
If s[i-1] != t[j-1]: dp[i][j] = dp[i-1][j]
  (must skip s[i-1])

Base Cases:
dp[i][0] = 1 for all i (empty t is a subsequence of any string)
dp[0][j] = 0 for j > 0 (non-empty t cannot be formed from empty s)

Step-by-Step: Larger Example

Step 1: Initialize first column to 1 (empty target matches everything) and first row (except dp[0][0]) to 0.
Step 2: Fill row by row. For each cell dp[i][j], check if s[i-1] == t[j-1].
Step 3: If characters match: we can either USE this character (dp[i-1][j-1]) or SKIP it (dp[i-1][j]). Sum both options.
Step 4: If characters don't match: we must skip s[i-1], so dp[i][j] = dp[i-1][j].
Step 5: Answer is dp[m][n] where m = len(s), n = len(t).

Code: Distinct Subsequences

// Distinct Subsequences
// Time: O(m*n), Space: O(m*n) or O(n) optimized
int numDistinct(string& s, string& t) {
    int m = s.size(), n = t.size();
    // dp[i][j] = ways to form t[0..j-1] from s[0..i-1]
    vector<vector<unsigned long long>> dp(m + 1, vector<unsigned long long>(n + 1, 0));

    // Base case: empty t matches any prefix of s
    for (int i = 0; i <= m; i++)
        dp[i][0] = 1;

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            dp[i][j] = dp[i - 1][j]; // skip s[i-1]
            if (s[i - 1] == t[j - 1])
                dp[i][j] += dp[i - 1][j - 1]; // use s[i-1]
        }
    }

    return dp[m][n];
}

// Space-optimized O(n) version
int numDistinctOpt(string& s, string& t) {
    int m = s.size(), n = t.size();
    vector<unsigned long long> dp(n + 1, 0);
    dp[0] = 1;

    for (int i = 1; i <= m; i++) {
        // Traverse right to left to use previous row values
        for (int j = n; j >= 1; j--) {
            if (s[i - 1] == t[j - 1])
                dp[j] += dp[j - 1];
        }
    }

    return dp[n];
}

Problem 2: Word Break

Given a string s and a dictionary of words wordDict, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

Example: s = "leetcode", wordDict = ["leet", "code"]
Answer: true ("leet" + "code")

Example: s = "applepenapple", wordDict = ["apple", "pen"]
Answer: true ("apple" + "pen" + "apple")

Example: s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
Answer: false (no valid segmentation)

State Definition: Word Break

State: dp[i] = whether s[0..i-1] can be segmented into dictionary words

Transition: dp[i] = true if there exists j in [0, i) such that dp[j] == true AND s[j..i-1] is in the dictionary

Base Case: dp[0] = true (empty string is always valid)

Answer: dp[n] where n = len(s)

Visual Walkthrough: Word Break

s = "leetcode", dict = {"leet", "code"}

Index:  0   1   2   3   4   5   6   7   8
dp:   [ T   F   F   F   T   F   F   F   T ]
             l   e   e   t   c   o   d   e

dp[0] = true (base case, empty string)
dp[1]: check s[0..0]="l" in dict? No  -> false
dp[2]: check s[0..1]="le" in dict? No -> false
dp[3]: check s[0..2]="lee" in dict? No -> false
dp[4]: check s[0..3]="leet" in dict? Yes! AND dp[0]=true -> TRUE
dp[5]: check s[0..4]="leetc"? No. s[4..4]="c"? dp[4]=T but "c" not in dict -> false
dp[6]: similar checks, all fail -> false
dp[7]: similar checks, all fail -> false
dp[8]: check s[4..7]="code" in dict? Yes! AND dp[4]=true -> TRUE

Answer: dp[8] = true

Step-by-Step: Word Break

Step 1: Put all dictionary words in a HashSet for O(1) lookup.
Step 2: Initialize dp[0] = true. All other entries start as false.
Step 3: For each position i from 1 to n, try all possible last words ending at position i. For each j from 0 to i-1, check if dp[j] is true AND s[j..i-1] is in the dictionary.
Step 4: Optimization: only check j values where i-j is at most the maximum word length in the dictionary.

Code: Word Break

// Word Break
// Time: O(n * m * k) where m = dict size, k = avg word length
// Optimized: O(n * maxWordLen)
// Space: O(n)
bool wordBreak(string s, vector<string>& wordDict) {
    int n = s.size();
    unordered_set<string> dict(wordDict.begin(), wordDict.end());

    // Find max word length for optimization
    int maxLen = 0;
    for (auto& w : wordDict)
        maxLen = max(maxLen, (int)w.size());

    vector<bool> dp(n + 1, false);
    dp[0] = true; // empty string

    for (int i = 1; i <= n; i++) {
        // Try all possible last words ending at position i
        for (int j = max(0, i - maxLen); j < i; j++) {
            if (dp[j] && dict.count(s.substr(j, i - j))) {
                dp[i] = true;
                break;
            }
        }
    }

    return dp[n];
}

// Word Break
// Time: O(n * maxWordLen), Space: O(n)
public boolean wordBreak(String s, List<String> wordDict) {
    int n = s.length();
    Set<String> dict = new HashSet<>(wordDict);

    int maxLen = 0;
    for (String w : wordDict)
        maxLen = Math.max(maxLen, w.length());

    boolean[] dp = new boolean[n + 1];
    dp[0] = true; // empty string

    for (int i = 1; i <= n; i++) {
        for (int j = Math.max(0, i - maxLen); j < i; j++) {
            if (dp[j] && dict.contains(s.substring(j, i))) {
                dp[i] = true;
                break;
            }
        }
    }

    return dp[n];
}

# Word Break
# Time: O(n * maxWordLen), Space: O(n)
def wordBreak(s: str, wordDict: list) -> bool:
    n = len(s)
    word_set = set(wordDict)
    max_len = max(len(w) for w in wordDict) if wordDict else 0

    dp = [False] * (n + 1)
    dp[0] = True  # empty string

    for i in range(1, n + 1):
        for j in range(max(0, i - max_len), i):
            if dp[j] and s[j:i] in word_set:
                dp[i] = True
                break

    return dp[n]

Complexity Analysis

ProblemTimeSpaceSpace (Optimized)
Distinct SubsequencesO(m * n)O(m * n)O(n)
Word BreakO(n * maxWordLen)O(n)O(n)
  • Distinct Subsequences: m = len(s), n = len(t). Space optimization uses right-to-left traversal on a 1D array.
  • Word Break: The maxWordLen optimization avoids checking substrings longer than any dictionary word, making it much faster in practice.
  • Overflow: Distinct subsequences can produce very large numbers. Use unsigned long long in C++ or handle modular arithmetic if needed.
  • Word Break II: To find ALL segmentations, use backtracking with memoization (returns list of sentences).

Key Insights

Distinct Subsequences: The key insight is the "use or skip" choice at each character of s. If characters match, we have two options; if not, we must skip.

Word Break: Think of it as: "can I reach position i from some valid position j by using a dictionary word?" This is like a reachability problem on a 1D graph.

Related Variations

1. Distinct Subsequences with k skips allowed
   - Add a dimension: dp[i][j][skips]

2. Word Break II (all segmentations)
   - Use dp to check feasibility, then backtrack for all paths

3. Concatenated Words
   - Word break where the "string" itself is a dictionary word
   - dp[i] = can s[0..i-1] be formed by 2+ dict words?

4. Interleaving String
   - Given s1, s2, s3: is s3 an interleaving of s1 and s2?
   - dp[i][j] = can s3[0..i+j-1] be formed by s1[0..i-1] and s2[0..j-1]?

Shortest Path - Complete Summary

A quick reference for choosing the right shortest path algorithm.

Algorithm Comparison Table

Algorithm Type Negative Wt? Time Best For
BFS Single source N/A (unweighted) O(V + E) Unweighted graphs
0-1 BFS Single source No O(V + E) Weights are only 0 or 1
DAG relaxation Single source Yes O(V + E) DAGs (no cycles)
Dijkstra Single source No O((V+E)logV) Non-negative weights
Bellman-Ford Single source Yes O(V × E) Negative weights, negative cycle detection
Floyd-Warshall All pairs Yes O(V³) All pairs, small graphs (V ≤ 400)
Multi-source BFS Multiple sources N/A O(V + E) Distance from nearest source

Decision Flowchart: Which Algorithm to Use?

Ask yourself these questions:

Q1: Is the graph unweighted?
→ YES: Use BFS (O(V+E), simplest)
Q2: Are all weights 0 or 1?
→ YES: Use 0-1 BFS with deque (O(V+E))
Q3: Is it a DAG?
→ YES: Use Topo sort + relaxation (O(V+E), handles negative weights too!)
Q4: Are all weights non-negative?
→ YES: Use Dijkstra (O((V+E)logV))
Q5: Has negative weights?
→ YES: Use Bellman-Ford (O(V×E))
Q6: Need ALL pairs shortest path?
→ YES: Use Floyd-Warshall (O(V³)) if V ≤ 400
→ For larger V: Run Dijkstra from each vertex

0-1 BFS

When edge weights are only 0 or 1, use a deque instead of priority queue. Push weight-0 edges to front, weight-1 edges to back. Achieves O(V+E) instead of O((V+E) log V).

Multi-Source BFS

Instead of starting BFS from one node, start from ALL source nodes simultaneously. Finds shortest distance to nearest source for every cell.

  Example: "Rotten Oranges" / "01 Matrix"

  Grid (1 = source, 0 = empty):
  0  0  0            2  1  2
  0  1  0    →       1  0  1     (distance to nearest 1)
  0  0  0            2  1  2

  Trick: Add ALL sources to queue at start (distance 0)
         Then BFS outward - each layer is distance+1

// Multi-source BFS on a grid
static int[][] multiSourceBFS(int[][] grid) {
    int R = grid.length, C = grid[0].length;
    int[][] dist = new int[R][C];
    for (int[] row : dist) Arrays.fill(row, -1);

    Queue<int[]> queue = new LinkedList<>();

    // Add ALL sources to queue (distance 0)
    for (int i = 0; i < R; i++)
        for (int j = 0; j < C; j++)
            if (grid[i][j] == 1) {
                queue.add(new int[]{i, j});
                dist[i][j] = 0;
            }

    int[][] dirs = {{0,1},{0,-1},{1,0},{-1,0}};

    while (!queue.isEmpty()) {
        int[] cell = queue.poll();
        for (int[] d : dirs) {
            int nr = cell[0] + d[0], nc = cell[1] + d[1];
            if (nr >= 0 && nr < R && nc >= 0 && nc < C && dist[nr][nc] == -1) {
                dist[nr][nc] = dist[cell[0]][cell[1]] + 1;
                queue.add(new int[]{nr, nc});
            }
        }
    }
    return dist;
}

// Multi-source BFS on a grid
vector<vector<int>> multiSourceBFS(vector<vector<int>>& grid) {
    int R = grid.size(), C = grid[0].size();
    vector<vector<int>> dist(R, vector<int>(C, -1));
    queue<pair<int,int>> q;

    // Add ALL sources
    for (int i = 0; i < R; i++)
        for (int j = 0; j < C; j++)
            if (grid[i][j] == 1) {
                q.push({i, j});
                dist[i][j] = 0;
            }

    int dx[] = {0,0,1,-1}, dy[] = {1,-1,0,0};

    while (!q.empty()) {
        auto [r, c] = q.front(); q.pop();
        for (int d = 0; d < 4; d++) {
            int nr = r + dx[d], nc = c + dy[d];
            if (nr >= 0 && nr < R && nc >= 0 && nc < C && dist[nr][nc] == -1) {
                dist[nr][nc] = dist[r][c] + 1;
                q.push({nr, nc});
            }
        }
    }
    return dist;
}

from collections import deque

def multi_source_bfs(grid):
    R, C = len(grid), len(grid[0])
    dist = [[-1] * C for _ in range(R)]
    queue = deque()

    # Add ALL sources to queue (distance 0)
    for i in range(R):
        for j in range(C):
            if grid[i][j] == 1:
                queue.append((i, j))
                dist[i][j] = 0

    dirs = [(0,1), (0,-1), (1,0), (-1,0)]

    while queue:
        r, c = queue.popleft()
        for dr, dc in dirs:
            nr, nc = r + dr, c + dc
            if 0 <= nr < R and 0 <= nc < C and dist[nr][nc] == -1:
                dist[nr][nc] = dist[r][c] + 1
                queue.append((nr, nc))
    return dist

Common Patterns & Tips

  • "Shortest path with at most K edges" → Bellman-Ford (run only K iterations)
  • "Shortest path in grid" → BFS (unweighted) or Dijkstra (weighted)
  • "Shortest path with constraints" → Modified Dijkstra with state = (node, constraint)
  • "Negative cycle?" → Bellman-Ford (check after V-1 iterations)
  • "Shortest path between all pairs, V ≤ 400" → Floyd-Warshall
  • "Shortest path, but can change at most 1 edge to 0" → Dijkstra from src + Dijkstra from dest, try each edge

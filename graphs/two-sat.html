2-SAT

2-SAT (2-Satisfiability) is the problem of determining whether a boolean formula in conjunctive normal form (CNF) with exactly 2 literals per clause can be satisfied. Unlike general SAT (NP-complete), 2-SAT is solvable in O(V + E) time using implication graphs and strongly connected components.

Problem Statement

Given a boolean formula like:

(x1 OR x2) AND (NOT x1 OR x3) AND (NOT x2 OR NOT x3) AND (x1 OR NOT x2)

Determine if there exists an assignment of true/false to variables that makes the entire formula true. If yes, find such an assignment.

Implication Graph

Each clause (a OR b) is equivalent to two implications:

  • NOT a ==> b (if a is false, b must be true)
  • NOT b ==> a (if b is false, a must be true)

We build a directed graph with 2n nodes (one for each variable and its negation) and add edges for these implications.

Visual Walkthrough

    Formula: (x1 OR x2) AND (NOT x1 OR x2) AND (x1 OR NOT x2)

    Clause 1: (x1 OR x2)
      NOT x1 ==> x2       (edge: ~x1 -> x2)
      NOT x2 ==> x1       (edge: ~x2 -> x1)

    Clause 2: (NOT x1 OR x2)
      x1 ==> x2           (edge: x1 -> x2)
      NOT x2 ==> NOT x1   (edge: ~x2 -> ~x1)

    Clause 3: (x1 OR NOT x2)
      NOT x1 ==> NOT x2   (edge: ~x1 -> ~x2)
      x2 ==> x1           (edge: x2 -> x1)

    Implication Graph:
    ┌──────────────────────────────┐
    │  x1 ──────> x2              │
    │  ^          ^               │
    │  |         /                │
    │  x2 ─────/                  │
    │                             │
    │  ~x1 ────> x2              │
    │  ~x1 ────> ~x2             │
    │  ~x2 ────> x1              │
    │  ~x2 ────> ~x1             │
    └──────────────────────────────┘

    SCCs: {x1, x2} and {~x1, ~x2}

    Since x1 and ~x1 are in DIFFERENT SCCs: SATISFIABLE!
    Solution: x1=true, x2=true (from SCC ordering)

    UNSATISFIABLE example:
    If x1 and ~x1 were in the SAME SCC, it would mean
    x1 ==> ~x1 ==> x1 (contradiction!)

Algorithm

Step 1: For each clause (a OR b), add edges: NOT a -> b and NOT b -> a to the implication graph.
Step 2: Find all Strongly Connected Components (SCCs) using Tarjan's or Kosaraju's algorithm.
Step 3: Check satisfiability: for each variable x, if x and NOT x are in the same SCC, the formula is UNSATISFIABLE (contradiction: x implies NOT x implies x).
Step 4: Extract solution: for each variable x, set x = true if comp[x] > comp[NOT x] (where comp[v] is the SCC number in reverse topological order). This ensures we pick the "later" implication.

Encoding Common Constraints

Constraint Clauses Implications
x must be true (x OR x) ~x -> x
x must be false (~x OR ~x) x -> ~x
At least one of x, y (x OR y) ~x->y, ~y->x
At most one of x, y (~x OR ~y) x->~y, y->~x
x and y same value (x OR ~y) AND (~x OR y) ~x->~y, y->x, x->y, ~y->~x
x and y different (x OR y) AND (~x OR ~y) ~x->y, ~y->x, x->~y, y->~x

Implementation

#include <bits/stdc++.h>
using namespace std;

class TwoSAT {
    int n; // number of variables
    vector<vector<int>> adj, radj;
    vector<int> order, comp;
    vector<bool> visited;

    void dfs1(int v) {
        visited[v] = true;
        for (int u : adj[v])
            if (!visited[u]) dfs1(u);
        order.push_back(v);
    }

    void dfs2(int v, int c) {
        comp[v] = c;
        for (int u : radj[v])
            if (comp[u] == -1) dfs2(u, c);
    }

public:
    TwoSAT(int n) : n(n), adj(2*n), radj(2*n), comp(2*n, -1), visited(2*n, false) {}

    // Variable x: positive literal = 2*x, negative = 2*x+1
    int pos(int x) { return 2 * x; }
    int neg(int x) { return 2 * x + 1; }

    // Add clause (a OR b) where a, b are literals
    void addClause(int a, int b) {
        // NOT a => b
        adj[a ^ 1].push_back(b);
        radj[b].push_back(a ^ 1);
        // NOT b => a
        adj[b ^ 1].push_back(a);
        radj[a].push_back(b ^ 1);
    }

    // Force variable x to be true
    void setTrue(int x) { addClause(pos(x), pos(x)); }

    // Force variable x to be false
    void setFalse(int x) { addClause(neg(x), neg(x)); }

    bool solve(vector<bool>& result) {
        // Kosaraju's SCC
        for (int i = 0; i < 2*n; i++)
            if (!visited[i]) dfs1(i);

        int c = 0;
        for (int i = 2*n - 1; i >= 0; i--)
            if (comp[order[i]] == -1) dfs2(order[i], c++);

        // Check satisfiability
        result.resize(n);
        for (int i = 0; i < n; i++) {
            if (comp[pos(i)] == comp[neg(i)]) return false;
            // x is true if comp[x] > comp[~x]
            result[i] = comp[pos(i)] > comp[neg(i)];
        }
        return true;
    }
};

int main() {
    // 3 variables: x0, x1, x2
    TwoSAT sat(3);

    // (x0 OR x1)
    sat.addClause(sat.pos(0), sat.pos(1));
    // (NOT x0 OR x1)
    sat.addClause(sat.neg(0), sat.pos(1));
    // (x0 OR NOT x2)
    sat.addClause(sat.pos(0), sat.neg(2));
    // (NOT x1 OR NOT x2)
    sat.addClause(sat.neg(1), sat.neg(2));

    vector<bool> result;
    if (sat.solve(result)) {
        cout << "SATISFIABLE" << endl;
        for (int i = 0; i < 3; i++)
            cout << "x" << i << " = " << (result[i] ? "true" : "false") << endl;
    } else {
        cout << "UNSATISFIABLE" << endl;
    }
    return 0;
}
import java.util.*;

class TwoSAT {
    int n;
    List<List<Integer>> adj, radj;
    int[] comp;
    boolean[] visited;
    List<Integer> order;

    TwoSAT(int n) {
        this.n = n;
        adj = new ArrayList<>();
        radj = new ArrayList<>();
        for (int i = 0; i < 2*n; i++) {
            adj.add(new ArrayList<>());
            radj.add(new ArrayList<>());
        }
        comp = new int[2*n];
        Arrays.fill(comp, -1);
        visited = new boolean[2*n];
        order = new ArrayList<>();
    }

    int pos(int x) { return 2 * x; }
    int neg(int x) { return 2 * x + 1; }

    void addClause(int a, int b) {
        adj.get(a ^ 1).add(b);
        radj.get(b).add(a ^ 1);
        adj.get(b ^ 1).add(a);
        radj.get(a).add(b ^ 1);
    }

    void dfs1(int v) {
        visited[v] = true;
        for (int u : adj.get(v))
            if (!visited[u]) dfs1(u);
        order.add(v);
    }

    void dfs2(int v, int c) {
        comp[v] = c;
        for (int u : radj.get(v))
            if (comp[u] == -1) dfs2(u, c);
    }

    boolean[] solve() {
        // Kosaraju's SCC
        for (int i = 0; i < 2*n; i++)
            if (!visited[i]) dfs1(i);

        int c = 0;
        for (int i = 2*n - 1; i >= 0; i--)
            if (comp[order.get(i)] == -1) dfs2(order.get(i), c++);

        boolean[] result = new boolean[n];
        for (int i = 0; i < n; i++) {
            if (comp[pos(i)] == comp[neg(i)]) return null;
            result[i] = comp[pos(i)] > comp[neg(i)];
        }
        return result;
    }

    public static void main(String[] args) {
        TwoSAT sat = new TwoSAT(3);

        // (x0 OR x1) AND (NOT x0 OR x1) AND (x0 OR NOT x2) AND (NOT x1 OR NOT x2)
        sat.addClause(sat.pos(0), sat.pos(1));
        sat.addClause(sat.neg(0), sat.pos(1));
        sat.addClause(sat.pos(0), sat.neg(2));
        sat.addClause(sat.neg(1), sat.neg(2));

        boolean[] result = sat.solve();
        if (result != null) {
            System.out.println("SATISFIABLE");
            for (int i = 0; i < 3; i++)
                System.out.println("x" + i + " = " + result[i]);
        } else {
            System.out.println("UNSATISFIABLE");
        }
    }

import sys
sys.setrecursionlimit(500000)

class TwoSAT:
    def __init__(self, n):
        """n = number of boolean variables"""
        self.n = n
        self.adj = [[] for _ in range(2 * n)]
        self.radj = [[] for _ in range(2 * n)]

    def pos(self, x):
        """Positive literal of variable x"""
        return 2 * x

    def neg(self, x):
        """Negative literal of variable x"""
        return 2 * x + 1

    def add_clause(self, a, b):
        """Add clause (a OR b) where a,b are literals from pos()/neg()"""
        # NOT a => b
        self.adj[a ^ 1].append(b)
        self.radj[b].append(a ^ 1)
        # NOT b => a
        self.adj[b ^ 1].append(a)
        self.radj[a].append(b ^ 1)

    def set_true(self, x):
        """Force variable x to be true"""
        self.add_clause(self.pos(x), self.pos(x))

    def set_false(self, x):
        """Force variable x to be false"""
        self.add_clause(self.neg(x), self.neg(x))

    def solve(self):
        """Returns list of booleans or None if unsatisfiable"""
        n2 = 2 * self.n
        visited = [False] * n2
        order = []
        comp = [-1] * n2

        # First pass: get finish order
        def dfs1(v):
            stack = [(v, 0)]
            visited[v] = True
            while stack:
                node, idx = stack[-1]
                if idx < len(self.adj[node]):
                    stack[-1] = (node, idx + 1)
                    nxt = self.adj[node][idx]
                    if not visited[nxt]:
                        visited[nxt] = True
                        stack.append((nxt, 0))
                else:
                    order.append(node)
                    stack.pop()

        # Second pass: assign components
        def dfs2(v, c):
            stack = [v]
            comp[v] = c
            while stack:
                node = stack.pop()
                for u in self.radj[node]:
                    if comp[u] == -1:
                        comp[u] = c
                        stack.append(u)

        for i in range(n2):
            if not visited[i]:
                dfs1(i)

        c = 0
        for i in range(n2 - 1, -1, -1):
            if comp[order[i]] == -1:
                dfs2(order[i], c)
                c += 1

        # Check and extract solution
        result = []
        for i in range(self.n):
            if comp[self.pos(i)] == comp[self.neg(i)]:
                return None  # UNSATISFIABLE
            result.append(comp[self.pos(i)] > comp[self.neg(i)])
        return result

# Example
sat = TwoSAT(3)

# (x0 OR x1) AND (NOT x0 OR x1) AND (x0 OR NOT x2) AND (NOT x1 OR NOT x2)
sat.add_clause(sat.pos(0), sat.pos(1))
sat.add_clause(sat.neg(0), sat.pos(1))
sat.add_clause(sat.pos(0), sat.neg(2))
sat.add_clause(sat.neg(1), sat.neg(2))

result = sat.solve()
if result:
    print("SATISFIABLE")
    for i, val in enumerate(result):
        print(f"x{i} = {val}")
else:
    print("UNSATISFIABLE")

Complexity Analysis

  • Time: O(n + m) where n = number of variables, m = number of clauses. Building the graph is O(m), SCC is O(V + E) = O(n + m).
  • Space: O(n + m) for the implication graph.

Applications

  • Scheduling: "Task A or Task B must be scheduled in slot 1" = clause (A_in_1 OR B_in_1).
  • Graph coloring with 2 colors: Each edge (u,v) means "u and v have different colors" = (u OR v) AND (NOT u OR NOT v).
  • Network configuration: "Server A uses protocol X or Server B uses protocol Y."
  • Game theory: Determining winning strategies in combinatorial games.
  • Circuit satisfiability: When circuits have fan-in 2 gates.
Comparison with general SAT:
  • 2-SAT: O(n + m), polynomial -- in P
  • 3-SAT: NP-complete (no known polynomial algorithm)
  • The jump from 2 to 3 literals per clause changes the problem from easy to hard!

Key Takeaways

  1. 2-SAT reduces to finding SCCs in the implication graph.
  2. Formula is satisfiable iff no variable and its negation share an SCC.
  3. The solution is extracted from the topological order of SCCs.
  4. Many constraint problems can be encoded as 2-SAT when each constraint involves at most 2 boolean variables.
  5. Linear time O(n + m) makes it practical for large inputs.

Graph Representation

A graph is a collection of vertices (nodes) connected by edges (links).

Types of Graphs

  UNDIRECTED GRAPH          DIRECTED GRAPH            WEIGHTED GRAPH

    1 --- 2                   1 → 2                   1 --5-- 2
    |     |                   ↓   ↓                   |       |
    |     |                   ↓   ↓                   3       7
    4 --- 3                   4 → 3                   |       |
                                                      4 --2-- 3
  • Undirected: Edges have no direction. If A connects to B, B also connects to A.
  • Directed: Edges have direction. A→B doesn't mean B→A.
  • Weighted: Each edge has a cost/weight associated with it.

1. Adjacency Matrix

A 2D array where matrix[i][j] = 1 means there's an edge from node i to node j.

  Graph:              Adjacency Matrix:
  0 --- 1              0  1  2  3
  |     |          0 [ 0  1  0  1 ]
  |     |          1 [ 1  0  1  0 ]
  3 --- 2          2 [ 0  1  0  1 ]
                   3 [ 1  0  1  0 ]
When to use: Dense graphs (many edges), need O(1) edge lookup.
Space: O(V²) — wasteful for sparse graphs.

2. Adjacency List

Each node stores a list of its neighbors. Most commonly used representation.

  Graph:              Adjacency List:
  0 --- 1            0 → [1, 3]
  |     |            1 → [0, 2]
  |     |            2 → [1, 3]
  3 --- 2            3 → [0, 2]
When to use: Sparse graphs (few edges), most competitive programming problems.
Space: O(V + E) — efficient!

Comparison

Operation Adjacency Matrix Adjacency List
Check if edge exists O(1) O(degree)
Find all neighbors O(V) O(degree)
Space O(V²) O(V + E)
Add edge O(1) O(1)

Code: Building a Graph (Adjacency List)

import java.util.*;

public class Graph {
    // Number of vertices
    int V;
    // Adjacency list: each index has a list of neighbors
    List<List<Integer>> adj;

    Graph(int V) {
        this.V = V;
        adj = new ArrayList<>();
        for (int i = 0; i < V; i++) {
            adj.add(new ArrayList<>());
        }
    }

    // Add edge for UNDIRECTED graph
    void addEdge(int u, int v) {
        adj.get(u).add(v);
        adj.get(v).add(u); // Remove this line for directed graph
    }

    public static void main(String[] args) {
        Graph g = new Graph(5);
        g.addEdge(0, 1);
        g.addEdge(0, 4);
        g.addEdge(1, 2);
        g.addEdge(1, 3);
        g.addEdge(3, 4);

        // Print adjacency list
        for (int i = 0; i < g.V; i++) {
            System.out.println(i + " → " + g.adj.get(i));
        }
    }
}

#include <bits/stdc++.h>
using namespace std;

int main() {
    int V = 5;
    // Adjacency list using vector of vectors
    vector<vector<int>> adj(V);

    // Add edge for UNDIRECTED graph
    auto addEdge = [&](int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u); // Remove for directed
    };

    addEdge(0, 1);
    addEdge(0, 4);
    addEdge(1, 2);
    addEdge(1, 3);
    addEdge(3, 4);

    // Print adjacency list
    for (int i = 0; i < V; i++) {
        cout << i << " → ";
        for (int x : adj[i]) cout << x << " ";
        cout << endl;
    }
    return 0;
}

Code: Weighted Graph (Adjacency List)

import java.util.*;

public class WeightedGraph {
    int V;
    // Each neighbor stored as {node, weight}
    List<List<int[]>> adj;

    WeightedGraph(int V) {
        this.V = V;
        adj = new ArrayList<>();
        for (int i = 0; i < V; i++)
            adj.add(new ArrayList<>());
    }

    void addEdge(int u, int v, int weight) {
        adj.get(u).add(new int[]{v, weight});
        adj.get(v).add(new int[]{u, weight}); // undirected
    }
}

#include <bits/stdc++.h>
using namespace std;

int main() {
    int V = 5;
    // {neighbor, weight}
    vector<vector<pair<int,int>>> adj(V);

    auto addEdge = [&](int u, int v, int w) {
        adj[u].push_back({v, w});
        adj[v].push_back({u, w}); // undirected
    };

    addEdge(0, 1, 5);
    addEdge(1, 2, 3);
    addEdge(0, 3, 7);
    return 0;
}

Edge List Representation

Simply store all edges as a list of tuples: (u, v, weight). Used in Kruskal's algorithm.

// Edge list: array of {u, v, weight}
int[][] edges = {
    {0, 1, 5},
    {1, 2, 3},
    {0, 3, 7},
    {2, 3, 2}
};

// Sort by weight (useful for Kruskal's)
Arrays.sort(edges, (a, b) -> a[2] - b[2]);

// Edge list: vector of {weight, u, v}
vector<tuple<int,int,int>> edges = {
    {5, 0, 1},
    {3, 1, 2},
    {7, 0, 3},
    {2, 2, 3}
};

// Sort by weight (useful for Kruskal's)
sort(edges.begin(), edges.end());

# Edge list: list of (u, v, weight)
edges = [
    (0, 1, 5),
    (1, 2, 3),
    (0, 3, 7),
    (2, 3, 2)
]

# Sort by weight (useful for Kruskal's)
edges.sort(key=lambda x: x[2])

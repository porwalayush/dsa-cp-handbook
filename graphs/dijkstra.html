Dijkstra's Algorithm

Dijkstra's algorithm finds the shortest path from a single source vertex to all other vertices in a weighted graph. It works only when all edge weights are non-negative.

How It Works (Intuition)

Think of it like spreading water from a source. Water always flows to the nearest unvisited node first. Once a node is "flooded," its shortest distance is finalized.

  1. Start at the source node with distance 0. All other nodes start at infinity.
  2. Pick the unvisited node with the smallest known distance (use a min-heap).
  3. For each neighbor of that node, check if going through the current node gives a shorter path. If yes, update the neighbor's distance.
  4. Mark the current node as visited (finalized).
  5. Repeat until all nodes are visited or the heap is empty.

Priority Queue Approach

We use a min-heap (priority queue) to always extract the node with the smallest tentative distance. This avoids scanning all nodes each time, making the algorithm efficient.

    Min-Heap operations during algorithm:

    Initial:   [(S, 0)]

    Pop S:     [(A, 1), (C, 3)]

    Pop A:     [(C, 3), (B, 5)]

    Pop C:     [(B, 5), (D, 6)]

    Pop B:     [(D, 6), (E, 12)]

    Pop D:     [(E, 7)]       <-- E updated from 12 to 7

    Pop E:     []             <-- Done!

Implementation

import java.util.*;

class Dijkstra {
    // Adjacency list: node -> list of (neighbor, weight)
    static int[] dijkstra(int V, List<List<int[]>> adj, int src) {
        // dist[i] = shortest distance from src to i
        int[] dist = new int[V];
        Arrays.fill(dist, Integer.MAX_VALUE);
        dist[src] = 0;

        // Min-heap: (distance, node)
        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
        pq.offer(new int[]{0, src});

        while (!pq.isEmpty()) {
            int[] curr = pq.poll();
            int d = curr[0], u = curr[1];

            // Skip if we already found a better path
            if (d > dist[u]) continue;

            // Relax all neighbors
            for (int[] edge : adj.get(u)) {
                int v = edge[0], w = edge[1];
                if (dist[u] + w < dist[v]) {
                    dist[v] = dist[u] + w;
                    pq.offer(new int[]{dist[v], v});
                }
            }
        }
        return dist;
    }

    public static void main(String[] args) {
        int V = 5;
        List<List<int[]>> adj = new ArrayList<>();
        for (int i = 0; i < V; i++) adj.add(new ArrayList<>());

        // Add edges: (from, to, weight)
        adj.get(0).add(new int[]{1, 4});  // S -> A, weight 4
        adj.get(0).add(new int[]{2, 1});  // S -> B, weight 1
        adj.get(2).add(new int[]{3, 2});  // B -> C, weight 2
        adj.get(3).add(new int[]{4, 3});  // C -> D, weight 3
        adj.get(1).add(new int[]{4, 6});  // A -> D, weight 6

        int[] dist = dijkstra(V, adj, 0);
        System.out.println(Arrays.toString(dist));
        // Output: [0, 4, 1, 3, 6]
    }
}

vector<int> dijkstra(int V, vector<vector<pair<int,int>>>& adj, int src) {
    // dist[i] = shortest distance from src to i
    vector<int> dist(V, INT_MAX);
    dist[src] = 0;

    // Min-heap: (distance, node)
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> pq;
    pq.push({0, src});

    while (!pq.empty()) {
        auto [d, u] = pq.top();
        pq.pop();

        // Skip if we already found a better path
        if (d > dist[u]) continue;

        // Relax all neighbors
        for (auto& [v, w] : adj[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}

import heapq

def dijkstra(V, adj, src):
    # adj[u] = [(v, weight), ...]
    dist = [float('inf')] * V
    dist[src] = 0
    # Min-heap: (distance, node)
    pq = [(0, src)]

    while pq:
        d, u = heapq.heappop(pq)
        # Skip if we already found a better path
        if d > dist[u]:
            continue
        # Relax all neighbors
        for v, w in adj[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))
    return dist

Time and Space Complexity

  • Time: O((V + E) log V) with a binary heap. Each vertex is extracted once (V log V) and each edge triggers at most one decrease-key / insert (E log V).
  • Space: O(V + E) for the adjacency list and distance array.
Important: Dijkstra's algorithm does NOT work with negative edge weights. If an edge has weight -3, Dijkstra might finalize a node's distance too early, missing a shorter path through the negative edge. Use Bellman-Ford for graphs with negative weights.

When to Use Dijkstra

  • Finding shortest paths in road networks, maps (all distances are positive).
  • Network routing protocols (OSPF uses Dijkstra).
  • Any graph where all edge weights are non-negative.
  • When you need single-source shortest path efficiently.

Key Takeaways

  1. Greedy approach: always pick the closest unvisited node.
  2. Once a node is popped from the min-heap, its distance is finalized.
  3. Non-negative weights only!
  4. Use adjacency list + priority queue for best performance.

BFS (Breadth-First Search)

BFS explores the graph level by level. It visits all neighbors of the current node before moving to the next level. Think of it like ripples spreading in water!

How BFS Works

   Start at node 0:

   Level 0:    [0]
                ↓
   Level 1:    [1] [3]
                ↓
   Level 2:    [2] [4]

   Graph:        BFS Order: 0 → 1 → 3 → 2 → 4
     0
    / \
   1   3
   |   |
   2   4

Algorithm Steps

  1. Put the starting node in a queue and mark it visited
  2. While queue is not empty:
    • Remove front node from queue
    • For each unvisited neighbor: mark visited, add to queue

Visual Walkthrough

Step 1: Queue = [0], Visited = {0}
Step 2: Pop 0 → add neighbors 1,3. Queue = [1,3], Visited = {0,1,3}
Step 3: Pop 1 → add neighbor 2. Queue = [3,2], Visited = {0,1,3,2}
Step 4: Pop 3 → add neighbor 4. Queue = [2,4], Visited = {0,1,3,2,4}
Step 5: Pop 2 → no new neighbors. Queue = [4]
Step 6: Pop 4 → no new neighbors. Queue = [] → DONE!

Key Properties

  • Time: O(V + E) — visits every vertex and edge once
  • Space: O(V) — for the queue and visited array
  • Shortest Path: BFS gives shortest path in UNWEIGHTED graphs!
  • Data Structure: Uses a Queue (FIFO)

Code: Basic BFS

import java.util.*;

public class BFS {
    static void bfs(int start, List<List<Integer>> adj, int V) {
        boolean[] visited = new boolean[V];
        Queue<Integer> queue = new LinkedList<>();

        visited[start] = true;
        queue.add(start);

        while (!queue.isEmpty()) {
            int node = queue.poll(); // remove front
            System.out.print(node + " ");

            // Visit all unvisited neighbors
            for (int neighbor : adj.get(node)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.add(neighbor);
                }
            }
        }
    }

    public static void main(String[] args) {
        int V = 5;
        List<List<Integer>> adj = new ArrayList<>();
        for (int i = 0; i < V; i++) adj.add(new ArrayList<>());

        // Add edges (undirected)
        adj.get(0).add(1); adj.get(1).add(0);
        adj.get(0).add(3); adj.get(3).add(0);
        adj.get(1).add(2); adj.get(2).add(1);
        adj.get(3).add(4); adj.get(4).add(3);

        System.out.print("BFS from 0: ");
        bfs(0, adj, V); // Output: 0 1 3 2 4
    }
}

BFS for Shortest Path (Unweighted)

// Returns shortest distance from source to all nodes
static int[] bfsShortestPath(int src, List<List<Integer>> adj, int V) {
    int[] dist = new int[V];
    Arrays.fill(dist, -1); // -1 means unreachable

    Queue<Integer> queue = new LinkedList<>();
    dist[src] = 0;
    queue.add(src);

    while (!queue.isEmpty()) {
        int node = queue.poll();
        for (int neighbor : adj.get(node)) {
            if (dist[neighbor] == -1) { // not visited
                dist[neighbor] = dist[node] + 1;
                queue.add(neighbor);
            }
        }
    }
    return dist;
}

BFS for Connected Components

// Count connected components in undirected graph
static int countComponents(List<List<Integer>> adj, int V) {
    boolean[] visited = new boolean[V];
    int count = 0;

    for (int i = 0; i < V; i++) {
        if (!visited[i]) {
            // BFS from this unvisited node
            Queue<Integer> q = new LinkedList<>();
            q.add(i);
            visited[i] = true;
            while (!q.isEmpty()) {
                int node = q.poll();
                for (int nb : adj.get(node)) {
                    if (!visited[nb]) {
                        visited[nb] = true;
                        q.add(nb);
                    }
                }
            }
            count++; // found one component
        }
    }
    return count;
}

int countComponents(vector<vector<int>>& adj, int V) {

            queue<int> q;
            q.push(i);
            visited[i] = true;
            while (!q.empty()) {
                int node = q.front(); q.pop();
                for (int nb : adj[node]) {

                        visited[nb] = true;
                        q.push(nb);
                    }
                }
            }
            count++;
        }
    }

}

from collections import deque

def count_components(adj, V):
    visited = [False] * V
    count = 0

    for i in range(V):
        if not visited[i]:
            queue = deque([i])
            visited[i] = True
            while queue:
                node = queue.popleft()
                for nb in adj[node]:
                    if not visited[nb]:
                        visited[nb] = True
                        queue.append(nb)
            count += 1
    return count

When to Use BFS?

  • Shortest path in unweighted graph
  • Level-order traversal (find nodes at distance k)
  • Connected components
  • Bipartite check (2-coloring)
  • Multi-source BFS (rotten oranges, 01-matrix)
  • When you need to explore closest nodes first

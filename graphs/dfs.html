DFS (Depth-First Search)

DFS explores the graph by going as deep as possible before backtracking. Think of it like exploring a maze — you go down one path until you hit a dead end, then come back and try another path.

How DFS Works

   Graph:            DFS Order (from 0): 0 → 1 → 2 → 3 → 4
     0
    / \
   1   3              Goes deep: 0 → 1 → 2 (dead end)
   |   |                         backtrack to 0 → 3 → 4
   2   4

Algorithm Steps

  1. Start at a node, mark it visited
  2. For each unvisited neighbor:
    • Recursively call DFS on that neighbor
  3. When no unvisited neighbors remain, backtrack

Visual Walkthrough

Step 1: Visit 0. Go to first unvisited neighbor: 1
Step 2: Visit 1. Go to first unvisited neighbor: 2
Step 3: Visit 2. No unvisited neighbors → BACKTRACK to 1 → BACKTRACK to 0
Step 4: Back at 0. Next unvisited neighbor: 3
Step 5: Visit 3. Go to neighbor: 4
Step 6: Visit 4. No unvisited neighbors → DONE!
Result: 0, 1, 2, 3, 4

DFS vs BFS

DFS

  • Uses Stack (or recursion)
  • Goes deep first
  • Good for: cycle detection, topological sort, pathfinding
  • May not find shortest path

BFS

  • Uses Queue
  • Goes wide first (level by level)
  • Good for: shortest path, level-order
  • Always finds shortest path (unweighted)

Code: Recursive DFS

import java.util.*;

public class DFS {
    static void dfs(int node, List<List<Integer>> adj, boolean[] visited) {
        visited[node] = true;
        System.out.print(node + " ");

        // Go to each unvisited neighbor (go DEEP)
        for (int neighbor : adj.get(node)) {
            if (!visited[neighbor]) {
                dfs(neighbor, adj, visited); // recursive call
            }
        }
        // When we reach here, we BACKTRACK
    }

    public static void main(String[] args) {
        int V = 5;
        List<List<Integer>> adj = new ArrayList<>();
        for (int i = 0; i < V; i++) adj.add(new ArrayList<>());

        adj.get(0).add(1); adj.get(1).add(0);
        adj.get(0).add(3); adj.get(3).add(0);
        adj.get(1).add(2); adj.get(2).add(1);
        adj.get(3).add(4); adj.get(4).add(3);

        boolean[] visited = new boolean[V];
        System.out.print("DFS from 0: ");
        dfs(0, adj, visited); // Output: 0 1 2 3 4
    }
}

Code: Iterative DFS (Using Stack)

static void dfsIterative(int start, List<List<Integer>> adj, int V) {
    boolean[] visited = new boolean[V];
    Stack<Integer> stack = new Stack<>();

    stack.push(start);

    while (!stack.isEmpty()) {
        int node = stack.pop();

        if (visited[node]) continue;
        visited[node] = true;
        System.out.print(node + " ");

        // Push neighbors to stack
        for (int neighbor : adj.get(node)) {
            if (!visited[neighbor]) {
                stack.push(neighbor);
            }
        }
    }
}

void dfsIterative(int start, vector<vector<int>>& adj, int V) {

}

DFS for Path Finding

// Check if path exists from src to dest
static boolean hasPath(int src, int dest, List<List<Integer>> adj, boolean[] visited) {
    if (src == dest) return true;

    visited[src] = true;
    for (int neighbor : adj.get(src)) {
        if (!visited[neighbor]) {
            if (hasPath(neighbor, dest, adj, visited))
                return true;
        }
    }
    return false;
}

def has_path(src, dest, adj, visited):
    if src == dest:
        return True

    visited[src] = True
    for neighbor in adj[src]:
        if not visited[neighbor]:
            if has_path(neighbor, dest, adj, visited):
                return True
    return False

Key Properties

  • Time: O(V + E)
  • Space: O(V) for visited array + O(V) recursion stack
  • Applications: Cycle detection, topological sort, SCC, bridges, articulation points
  • Caution: For very deep graphs, recursion may cause stack overflow → use iterative version

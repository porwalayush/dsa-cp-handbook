Bridges & Articulation Points

A bridge is an edge whose removal disconnects the graph. An articulation point (or cut vertex) is a vertex whose removal (along with its edges) disconnects the graph. These represent critical vulnerabilities in a network.

Real-world analogy: Imagine a road network. A bridge is a road that, if destroyed, splits the city into disconnected parts. An articulation point is an intersection that, if blocked, does the same.

Visual Example

    Graph:
                                    BRIDGE
        0 --- 1 --- 2 --- 3 ========= 4 --- 5
              |   /                          |   \
              |  /                           |    \
              3                              7 --- 6

    Wait, let me draw a cleaner example:

        1 --- 2 --- 3
        |   /       |
        |  /        |           BRIDGE: edge 3-4
        0       4 ---+--- 5     (removing it disconnects {0,1,2,3} from {4,5,6})
                |         |
                +--- 6 ---+

    Even cleaner:

        0 ---- 1 ---- 2
        |      |
        |      |         BRIDGE: edge 1--3
        3 ---- 4         (removing 1--3 disconnects the graph? No...)

    Correct clean example:

        0 --- 1 --- 3 --- 4
        |   /             |
        | /               |
        2                 5

    Bridges: 1---3 and 4---5 and 3---4
    Articulation Points: 1, 3, 4

    Why 1 is articulation point:
      Remove vertex 1 -> {0, 2} disconnected from {3, 4, 5}

    Why edge 1-3 is a bridge:
      Remove edge 1-3 -> {0, 1, 2} disconnected from {3, 4, 5}

Detailed Example with Discovery and Low Values

    Graph:

        0 --- 1 --- 3 --- 4
        |   /             |
        | /               |
        2                 5

    DFS Tree (starting from 0):

        0 (disc=0, low=0)
        |
        1 (disc=1, low=0)    <-- low=0 because 2 has back edge to 0
       / \
      2    3 (disc=3, low=3)  <-- low stays 3 (no back edge to anything earlier)
      |    |
  (back   4 (disc=4, low=4)
   to 0)  |
          5 (disc=5, low=4)  <-- low=4 because back edge to 4

    disc=2 for vertex 2, low=0 (back edge to 0)

    Finding BRIDGES:
      Edge u-v is a bridge if: low[v] > disc[u]

      Edge 1-3: low[3]=3 > disc[1]=1  => BRIDGE!
        (vertex 3 cannot reach anything above vertex 1 without using edge 1-3)

      Edge 0-1: low[1]=0, disc[0]=0  => NOT bridge (0 = 0, not greater)
      Edge 0-2: low[2]=0, disc[0]=0  => NOT bridge

    Finding ARTICULATION POINTS:
      For root: articulation point if it has 2+ children in DFS tree
      For non-root u: articulation point if any child v has low[v] >= disc[u]

      Vertex 1: child 3 has low[3]=3 >= disc[1]=1  => ARTICULATION POINT
      Vertex 3: child 4 has low[4]=4 >= disc[3]=3  => ARTICULATION POINT
      Vertex 0 (root): only 1 child in DFS tree => NOT articulation point

The Algorithm (Tarjan's Approach)

Key Concepts:
  • disc[u]: Discovery time — when vertex u was first visited in DFS
  • low[u]: The lowest discovery time reachable from the subtree of u (using back edges)
  • A back edge to an ancestor means there is an alternative path (cycle), so the edge is NOT critical
Bridge Condition:

Edge (u, v) is a bridge if low[v] > disc[u]

Meaning: vertex v (and its subtree) has NO back edge to u or any ancestor of u. The only way to reach v from above is through edge (u,v).

Articulation Point Conditions:
  • Root vertex: Is an articulation point if it has 2 or more children in the DFS tree
  • Non-root vertex u: Is an articulation point if there exists a child v such that low[v] >= disc[u] (no back edge from v's subtree to above u)

Implementation

import java.util.*;

public class BridgesAndAP {
    private int V;
    private List<List<Integer>> adj;
    private int timer = 0;

    public BridgesAndAP(int v) {
        V = v;
        adj = new ArrayList<>();
        for (int i = 0; i < V; i++)
            adj.add(new ArrayList<>());
    }

    public void addEdge(int u, int v) {
        adj.get(u).add(v);
        adj.get(v).add(u);
    }

    public void findBridgesAndAP() {
        int[] disc = new int[V];
        int[] low = new int[V];
        boolean[] visited = new boolean[V];
        int[] parent = new int[V];
        boolean[] isAP = new boolean[V];
        List<int[]> bridges = new ArrayList<>();

        Arrays.fill(parent, -1);

        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                dfs(i, disc, low, visited, parent, isAP, bridges);
            }
        }

        // Print bridges
        System.out.println("Bridges:");
        for (int[] bridge : bridges) {
            System.out.println(bridge[0] + " - " + bridge[1]);
        }

        // Print articulation points
        System.out.println("Articulation Points:");
        for (int i = 0; i < V; i++) {
            if (isAP[i]) System.out.println(i);
        }
    }

    private void dfs(int u, int[] disc, int[] low,
                     boolean[] visited, int[] parent,
                     boolean[] isAP, List<int[]> bridges) {
        visited[u] = true;
        disc[u] = low[u] = timer++;
        int children = 0; // Count DFS tree children (for root check)

        for (int v : adj.get(u)) {
            if (!visited[v]) {
                children++;
                parent[v] = u;
                dfs(v, disc, low, visited, parent, isAP, bridges);

                // After DFS of child, update low value
                low[u] = Math.min(low[u], low[v]);

                // BRIDGE check: low[v] > disc[u]
                if (low[v] > disc[u]) {
                    bridges.add(new int[]{u, v});
                }

                // ARTICULATION POINT check (non-root)
                if (parent[u] != -1 && low[v] >= disc[u]) {
                    isAP[u] = true;
                }
            } else if (v != parent[u]) {
                // Back edge: update low value
                low[u] = Math.min(low[u], disc[v]);
            }
        }

        // ARTICULATION POINT check (root with 2+ children)
        if (parent[u] == -1 && children >= 2) {
            isAP[u] = true;
        }
    }

    public static void main(String[] args) {
        BridgesAndAP g = new BridgesAndAP(6);
        g.addEdge(0, 1);
        g.addEdge(1, 2);
        g.addEdge(2, 0); // Triangle: 0-1-2 (no bridges here)
        g.addEdge(1, 3); // Bridge: 1-3
        g.addEdge(3, 4);
        g.addEdge(4, 5);
        g.addEdge(5, 3); // Triangle: 3-4-5 (no bridges here)

        g.findBridgesAndAP();
        // Bridges: 1-3
        // Articulation Points: 1, 3
    }
}
                
#include <bits/stdc++.h>
using namespace std;

class BridgesAndAP {
    int V, timer = 0;
    vector<vector<int>> adj;
    vector<int> disc, low, parent;
    vector<bool> visited, isAP;
    vector<pair<int,int>> bridges;

    void dfs(int u) {
        visited[u] = true;
        disc[u] = low[u] = timer++;
        int children = 0;

        for (int v : adj[u]) {
            if (!visited[v]) {
                children++;
                parent[v] = u;
                dfs(v);

                // Update low value from child
                low[u] = min(low[u], low[v]);

                // BRIDGE check
                if (low[v] > disc[u]) {
                    bridges.push_back({u, v});
                }

                // ARTICULATION POINT (non-root)
                if (parent[u] != -1 && low[v] >= disc[u]) {
                    isAP[u] = true;
                }
            } else if (v != parent[u]) {
                // Back edge: update low value
                low[u] = min(low[u], disc[v]);
            }
        }

        // ARTICULATION POINT (root with 2+ children)
        if (parent[u] == -1 && children >= 2) {
            isAP[u] = true;
        }
    }

public:
    BridgesAndAP(int v) : V(v), adj(v), disc(v), low(v),
        parent(v, -1), visited(v, false), isAP(v, false) {}

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    void findAll() {
        for (int i = 0; i < V; i++) {
            if (!visited[i]) dfs(i);
        }

        cout << "Bridges:" << endl;
        for (auto& [u, v] : bridges) {
            cout << u << " - " << v << endl;
        }

        cout << "Articulation Points:" << endl;
        for (int i = 0; i < V; i++) {
            if (isAP[i]) cout << i << endl;
        }
    }
};

int main() {
    BridgesAndAP g(6);
    g.addEdge(0, 1);
    g.addEdge(1, 2);
    g.addEdge(2, 0); // Triangle: 0-1-2
    g.addEdge(1, 3); // Bridge: 1-3
    g.addEdge(3, 4);
    g.addEdge(4, 5);
    g.addEdge(5, 3); // Triangle: 3-4-5

    g.findAll();
    // Bridges: 1 - 3
    // Articulation Points: 1, 3
    return 0;
}
                
def find_bridges(V, adj):
    disc = [-1] * V
    low = [-1] * V
    bridges = []
    timer = [0]

    def dfs(node, parent):
        disc[node] = low[node] = timer[0]
        timer[0] += 1

        for neighbor in adj[node]:
            if disc[neighbor] == -1:  # not visited
                dfs(neighbor, node)
                low[node] = min(low[node], low[neighbor])
                # Bridge condition
                if low[neighbor] > disc[node]:
                    bridges.append((node, neighbor))
            elif neighbor != parent:
                low[node] = min(low[node], disc[neighbor])

    for i in range(V):
        if disc[i] == -1:
            dfs(i, -1)

    return bridges

# Example
V = 5
adj = [[] for _ in range(V)]
edges = [(0,1), (1,2), (2,0), (1,3), (3,4)]
for u, v in edges:
    adj[u].append(v)
    adj[v].append(u)
print("Bridges:", find_bridges(V, adj))  # [(1,3), (3,4)]

Understanding Low-Link Values

    Complete walkthrough for this graph:

        0 --- 1 --- 3 --- 4
        |   /       |   /
        | /         | /
        2           5

    DFS from 0:

    Visit 0: disc[0]=0, low[0]=0
      Visit 1: disc[1]=1, low[1]=1
        Visit 2: disc[2]=2, low[2]=2
          Back edge 2->0: low[2] = min(2, disc[0]) = min(2,0) = 0
        Return to 1: low[1] = min(1, low[2]) = min(1,0) = 0
        Visit 3: disc[3]=3, low[3]=3
          Visit 4: disc[4]=4, low[4]=4
            Visit 5: disc[5]=5, low[5]=5
              Back edge 5->3: low[5] = min(5, disc[3]) = min(5,3) = 3
              Back edge 5->4: already parent, skip
            Return to 4: low[4] = min(4, low[5]) = min(4,3) = 3
            Back edge 4->3: 3 is parent, skip
          Return to 3: low[3] = min(3, low[4]) = min(3,3) = 3
        Return to 1: low[1] stays 0
        Check edge 1-3: low[3]=3 > disc[1]=1  => BRIDGE!
        Check AP for 1: low[3]=3 >= disc[1]=1 => YES, AP!
      Return to 0: low[0] = min(0, low[1]) = 0
      0 is root with 1 child => NOT AP

    Results:
      Bridge: 1-3
      Articulation Point: 1, 3

Complexity

  • Time: O(V + E) — single DFS traversal
  • Space: O(V + E) for the graph and auxiliary arrays

Applications

  • Network reliability: Identify single points of failure in computer networks
  • Infrastructure planning: Find critical roads/cables whose failure isolates parts of a network
  • Biconnected components: Decompose graph into 2-edge-connected or 2-vertex-connected components
  • Circuit design: Identify critical connections in electrical circuits
Interview Tip: The bridge/AP algorithm is essentially the same as Tarjan's SCC but for undirected graphs. The core idea (disc and low arrays + DFS) is shared. Master one, and the other comes naturally!

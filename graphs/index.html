Graph Basics & Traversals

Graph Representation

Adjacency Matrix, Adjacency List, Edge List. Directed vs Undirected. Weighted vs Unweighted.

Easy O(V+E) list

BFS (Breadth-First Search)

Level-order traversal using queue. Shortest path in unweighted graph. Connected components.

Easy O(V+E)

DFS (Depth-First Search)

Stack-based deep exploration. Backtracking. Discovery & finish times. DFS tree edges.

Easy O(V+E)
2

Cycle Detection

Cycle in Undirected Graph

Using DFS with parent tracking. Using DSU (Union-Find). Detecting back edges.

Easy O(V+E)

Cycle in Directed Graph

Using DFS coloring (White-Gray-Black). Using Kahn's BFS. Back edge detection.

Medium O(V+E)
3

Topological Sort & DAG

Topological Sort (DFS)

Linear ordering of vertices in DAG. Using DFS + stack. Applications in scheduling.

Medium O(V+E)

Kahn's Algorithm (BFS)

Topological sort using in-degree + queue. Also detects cycles. Level-wise processing.

Medium O(V+E)

DAG Shortest Path

Shortest path in DAG using topological order. Relaxation. Longest path in DAG.

Medium O(V+E)
4

Shortest Path Algorithms

Dijkstra's Algorithm

Single source shortest path. Greedy + Priority Queue. Non-negative weights only.

Medium O((V+E)logV)

Bellman-Ford Algorithm

Handles negative weights. Detects negative cycles. Relax all edges V-1 times.

Medium O(V*E)

Floyd-Warshall Algorithm

All-pairs shortest path. Dynamic Programming on intermediate vertices.

Medium O(V^3)

0-1 BFS

Shortest path when edge weights are 0 or 1. Uses deque instead of priority queue.

Medium O(V+E)

Shortest Path Summary

When to use which algorithm. Multi-source BFS. Comparison table of all methods.

Medium

Minimum Spanning Tree

Prim's Algorithm

Grow MST from a vertex. Greedy + Priority Queue. Dense graph friendly.

Medium O((V+E)logV)

Kruskal's Algorithm

Sort edges + Union-Find. Sparse graph friendly. Forest merging approach.

Medium O(E logE)
6

Disjoint Set Union (DSU)

DSU / Union-Find

Path compression + Union by rank. Near O(1) amortized. Dynamic connectivity.

Medium O(α(n)) amortized

Strongly Connected Components

Kosaraju's & Tarjan's algorithms. Condensation graph. 2-SAT connection.

Bridges & Articulation Points

Critical edges and vertices. Tarjan's DFS-based approach. Low-link values.

Bipartite Graph & Coloring

2-colorable check via BFS/DFS. Bipartite matching. Odd cycle detection.

Medium O(V+E)

2-SAT

Boolean satisfiability with 2 variables per clause. Implication graph + SCC.

Tree Algorithms

LCA (Lowest Common Ancestor)

Binary lifting. Euler tour + RMQ. O(n log n) preprocessing, O(log n) query.

Hard O(log n) query

Centroid Decomposition

Divide and conquer on trees. Path queries. Tree height reduces to O(log n).

Hard O(n log n)

Euler Tour & Flattening

Flatten tree to array. Subtree queries → range queries. Euler tour/path distinction.

Hard O(n)

Which Algorithm to Use?

Quick decision guide for graph problems

1

Shortest Path?

Unweighted→BFS. Non-negative→Dijkstra. Negative→Bellman-Ford. All-pairs→Floyd-Warshall. 0/1 weights→0-1 BFS.

2

Connectivity?

Static→BFS/DFS/DSU. Dynamic→DSU. Directed→SCC (Tarjan/Kosaraju). Critical→Bridges/APs.

3

Ordering/Scheduling?

Topological Sort (DFS or Kahn's). Check for cycles first. Lexicographically smallest→Kahn's+PQ.

4

Minimum Cost Tree?

Dense graph→Prim's. Sparse→Kruskal's. Both use greedy. Need DSU for Kruskal's.

5

Matching/Flow?

Bipartite matching→augmenting paths. Max flow→Dinic's/Edmonds-Karp. Min cut = max flow.

6

Tree Queries?

Ancestor→LCA (binary lifting). Path→LCA+difference. Subtree→Euler tour+segment tree.

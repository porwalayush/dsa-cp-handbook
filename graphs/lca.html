Lowest Common Ancestor (LCA)

The Lowest Common Ancestor of two nodes u and v in a rooted tree is the deepest node that is an ancestor of both u and v. LCA is fundamental for tree path queries and has multiple efficient solutions.

Problem Statement

Given a rooted tree with n nodes, answer queries of the form: "What is the LCA of nodes u and v?" Preprocess the tree so each query can be answered efficiently.

Approaches

Method Preprocess Query Space
Naive (climb up) O(n) O(n) O(n)
Binary Lifting O(n log n) O(log n) O(n log n)
Euler Tour + Sparse Table O(n log n) O(1) O(n log n)
Farach-Colton & Bender O(n) O(1) O(n)

Binary Lifting

The idea is to precompute for each node its ancestor at distance 2^0, 2^1, 2^2, ... Using these "jumps," we can reach any ancestor in O(log n) steps.

Preprocess: Build table up[v][k] = 2^k-th ancestor of v. Base: up[v][0] = parent[v]. Recurrence: up[v][k] = up[up[v][k-1]][k-1].
Query LCA(u, v): 1) Make u and v same depth by lifting the deeper one. 2) If they are the same node, that is the LCA. 3) Otherwise, lift both simultaneously using the largest jumps that keep them different. The parent of where they end up is the LCA.

Implementation (Binary Lifting)

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
const int LOG = 17; // log2(MAXN)

vector<int> adj[MAXN];
int up[MAXN][LOG]; // up[v][k] = 2^k-th ancestor of v
int depth[MAXN];
int n;

void dfs(int v, int parent, int d) {
    depth[v] = d;
    up[v][0] = parent;
    for (int k = 1; k < LOG; k++)
        up[v][k] = up[up[v][k-1]][k-1];

    for (int u : adj[v]) {
        if (u != parent)
            dfs(u, v, d + 1);
    }
}

int lca(int u, int v) {
    // Step 1: Make depths equal
    if (depth[u] < depth[v]) swap(u, v);
    int diff = depth[u] - depth[v];
    for (int k = 0; k < LOG; k++) {
        if ((diff >> k) & 1)
            u = up[u][k];
    }

    // Step 2: If same node, return
    if (u == v) return u;

    // Step 3: Lift both until they meet
    for (int k = LOG - 1; k >= 0; k--) {
        if (up[u][k] != up[v][k]) {
            u = up[u][k];
            v = up[v][k];
        }
    }
    return up[u][0]; // parent is the LCA
}

// Distance between two nodes
int dist(int u, int v) {
    return depth[u] + depth[v] - 2 * depth[lca(u, v)];
}

int main() {
    n = 9;
    // Build tree (1-indexed)
    adj[1] = {2, 3, 4};
    adj[2] = {1, 5, 6};
    adj[4] = {1, 7};
    adj[5] = {2, 8, 9};
    adj[3] = {1}; adj[6] = {2};
    adj[7] = {4}; adj[8] = {5}; adj[9] = {5};

    dfs(1, 1, 0); // root = 1, parent of root = itself

    cout << lca(8, 9) << endl; // 5
    cout << lca(8, 6) << endl; // 2
    cout << lca(5, 7) << endl; // 1
    cout << dist(8, 7) << endl; // 5
    return 0;
}
import java.util.*;

class LCA {
    static final int LOG = 17;
    static int[][] up;
    static int[] depth;
    static List<List<Integer>> adj;

    static void dfs(int v, int parent, int d) {
        depth[v] = d;
        up[v][0] = parent;
        for (int k = 1; k < LOG; k++)
            up[v][k] = up[up[v][k-1]][k-1];

        for (int u : adj.get(v)) {
            if (u != parent)
                dfs(u, v, d + 1);
        }
    }

    static int lca(int u, int v) {
        // Make depths equal
        if (depth[u] < depth[v]) { int t = u; u = v; v = t; }
        int diff = depth[u] - depth[v];
        for (int k = 0; k < LOG; k++) {
            if (((diff >> k) & 1) == 1)
                u = up[u][k];
        }
        if (u == v) return u;

        // Lift both
        for (int k = LOG - 1; k >= 0; k--) {
            if (up[u][k] != up[v][k]) {
                u = up[u][k];
                v = up[v][k];
            }
        }
        return up[u][0];
    }

    static int dist(int u, int v) {
        return depth[u] + depth[v] - 2 * depth[lca(u, v)];
    }

    public static void main(String[] args) {
        int n = 10; // 1-indexed, so allocate n+1
        up = new int[n][LOG];
        depth = new int[n];
        adj = new ArrayList<>();
        for (int i = 0; i < n; i++) adj.add(new ArrayList<>());

        // Build tree (1-indexed)
        int[][] edges = {{1,2},{1,3},{1,4},{2,5},{2,6},{4,7},{5,8},{5,9}};
        for (int[] e : edges) {
            adj.get(e[0]).add(e[1]);
            adj.get(e[1]).add(e[0]);
        }

        dfs(1, 1, 0);

        System.out.println(lca(8, 9)); // 5
        System.out.println(lca(8, 6)); // 2
        System.out.println(lca(5, 7)); // 1
        System.out.println(dist(8, 7)); // 5
    }
}
import sys
sys.setrecursionlimit(200000)

LOG = 17

def build_lca(n, adj, root=1):
    """Preprocess tree for LCA queries using binary lifting"""
    up = [[0] * LOG for _ in range(n + 1)]
    depth = [0] * (n + 1)

    def dfs(v, parent, d):
        depth[v] = d
        up[v][0] = parent
        for k in range(1, LOG):
            up[v][k] = up[up[v][k-1]][k-1]
        for u in adj[v]:
            if u != parent:
                dfs(u, v, d + 1)

    dfs(root, root, 0)
    return up, depth

def lca(u, v, up, depth):
    """Find LCA of u and v"""
    # Make depths equal
    if depth[u] < depth[v]:
        u, v = v, u
    diff = depth[u] - depth[v]
    for k in range(LOG):
        if (diff >> k) & 1:
            u = up[u][k]

    if u == v:
        return u

    # Lift both
    for k in range(LOG - 1, -1, -1):
        if up[u][k] != up[v][k]:
            u = up[u][k]
            v = up[v][k]
    return up[u][0]

def dist(u, v, up, depth):
    """Distance between u and v in tree"""
    return depth[u] + depth[v] - 2 * depth[lca(u, v, up, depth)]

# Example
n = 9
adj = [[] for _ in range(n + 1)]
edges = [(1,2),(1,3),(1,4),(2,5),(2,6),(4,7),(5,8),(5,9)]
for u, v in edges:
    adj[u].append(v)
    adj[v].append(u)

up, depth = build_lca(n, adj, root=1)

print(lca(8, 9, up, depth))  # 5
print(lca(8, 6, up, depth))  # 2
print(lca(5, 7, up, depth))  # 1
print(dist(8, 7, up, depth)) # 5

Euler Tour + Sparse Table (O(1) Query)

An alternative approach that achieves O(1) per query:

  1. Perform Euler tour of the tree. Record the sequence of nodes visited and their depths.
  2. LCA(u, v) = node with minimum depth between the first occurrences of u and v in the Euler tour.
  3. Use a Sparse Table for Range Minimum Query (RMQ) to answer in O(1).

Applications

  • Distance between nodes: dist(u, v) = depth[u] + depth[v] - 2 * depth[LCA(u,v)]
  • Path queries: Sum/max/min on path from u to v splits at LCA.
  • Tree isomorphism: Compare subtree structures.
  • Tarjan's offline LCA: Answer all queries in O(n * alpha(n)) using DSU.
  • Virtual tree: Build compressed tree containing only query nodes and their LCAs.

Complexity Analysis

  • Binary Lifting: O(n log n) preprocess, O(log n) per query, O(n log n) space.
  • Euler Tour + Sparse Table: O(n log n) preprocess, O(1) per query, O(n log n) space.
Practical tip: Binary lifting is the most common choice in competitive programming due to its simplicity and the ability to combine it with other information (e.g., maximum edge weight on path). Use Euler Tour + Sparse Table when O(1) per query is critical with many queries.

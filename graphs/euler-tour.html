Euler Tour / Euler Path

Visit every EDGE exactly once. (Hamiltonian = visit every vertex once, which is NP-hard. Euler is polynomial!)

Euler Path vs Euler Circuit

  EULER PATH: Visit every edge once. Start and end can be different.
  EULER CIRCUIT: Visit every edge once AND return to start.

  Example (Euler Circuit):        Example (Euler Path only):
      0 --- 1                         0 --- 1
      |  X  |                         |     |
      3 --- 2                         3 --- 2 --- 4

  Circuit: 0→1→2→3→0→2→1→3          Path: 0→1→2→3→0 (but no circuit
  (all edges used, back to start)     because degrees differ)

Conditions for Existence

Undirected Graph Directed Graph
Euler Circuit All vertices have EVEN degree Every vertex: in-degree = out-degree
Euler Path Exactly 0 or 2 vertices with ODD degree At most one vertex: out-in = 1 (start)
At most one vertex: in-out = 1 (end)
All others: in = out
Remember: Graph must also be connected (considering only vertices with edges)!

Hierholzer's Algorithm

Finds Euler path/circuit in O(E) time.

  1. Start from a valid starting vertex
  2. Follow edges (removing them as you go) until you get stuck
  3. If stuck at a vertex with remaining edges, start a sub-tour from there
  4. Splice sub-tours together
  Simplified approach (using stack):

  1. Push start vertex to stack
  2. While stack not empty:
     - curr = top of stack
     - If curr has unused edges:
         Pick one edge (curr → next), remove it, push next
     - Else:
         Pop curr, add to result (front)
  3. Reverse result = Euler path!

Code: Hierholzer's (Undirected Graph)

import java.util.*;

public class EulerPath {
    static List<Integer> findEulerPath(int V, List<List<int[]>> adj) {
        // adj[u] contains {v, edgeIndex} pairs
        // Find start: vertex with odd degree (or any if circuit)
        int start = 0;
        for (int i = 0; i < V; i++)
            if (adj.get(i).size() % 2 == 1) { start = i; break; }

        Stack<Integer> stack = new Stack<>();
        List<Integer> result = new ArrayList<>();
        boolean[] usedEdge = new boolean[/* total edges */];
        int[] ptr = new int[V]; // pointer to next unused edge for each vertex

        stack.push(start);
        while (!stack.isEmpty()) {
            int v = stack.peek();
            boolean found = false;

            while (ptr[v] < adj.get(v).size()) {
                int[] edge = adj.get(v).get(ptr[v]);
                ptr[v]++;
                if (!usedEdge[edge[1]]) {
                    usedEdge[edge[1]] = true;
                    stack.push(edge[0]);
                    found = true;
                    break;
                }
            }

            if (!found) {
                result.add(stack.pop());
            }
        }
        Collections.reverse(result);
        return result;
    }
}

#include <bits/stdc++.h>
using namespace std;

// Simpler version for directed graph (Euler circuit)
vector<int> findEulerCircuit(int V, vector<vector<int>>& adj) {
    vector<int> ptr(V, 0); // next edge to try
    stack<int> st;
    vector<int> result;

    st.push(0);
    while (!st.empty()) {
        int v = st.top();
        if (ptr[v] < (int)adj[v].size()) {
            int u = adj[v][ptr[v]++];
            st.push(u);
        } else {
            result.push_back(v);
            st.pop();
        }
    }
    reverse(result.begin(), result.end());
    return result;
}

// For undirected: need to track used edges
vector<int> findEulerPathUndirected(int V, vector<vector<pair<int,int>>>& adj, int E) {
    // adj[u] = {{v, edge_id}, ...}
    vector<bool> usedEdge(E, false);
    vector<int> ptr(V, 0);

    // Find start (odd degree vertex, or 0)
    int start = 0;
    for (int i = 0; i < V; i++)
        if (adj[i].size() % 2 == 1) { start = i; break; }

    stack<int> st;
    vector<int> result;
    st.push(start);

    while (!st.empty()) {
        int v = st.top();
        bool found = false;
        while (ptr[v] < (int)adj[v].size()) {
            auto [u, eid] = adj[v][ptr[v]++];
            if (!usedEdge[eid]) {
                usedEdge[eid] = true;
                st.push(u);
                found = true;
                break;
            }
        }
        if (!found) { result.push_back(st.top()); st.pop(); }
    }
    reverse(result.begin(), result.end());
    return result;
}

from collections import defaultdict

def find_euler_path(V, edges):
    # Build adjacency list with edge indices
    adj = defaultdict(list)
    for i, (u, v) in enumerate(edges):
        adj[u].append((v, i))
        adj[v].append((u, i))

    # Find start (odd degree vertex, or 0)
    start = 0
    for i in range(V):
        if len(adj[i]) % 2 == 1:
            start = i
            break

    used_edge = [False] * len(edges)
    stack = [start]
    result = []

    while stack:
        v = stack[-1]
        found = False
        while adj[v]:
            u, eid = adj[v].pop()
            if not used_edge[eid]:
                used_edge[eid] = True
                stack.append(u)
                found = True
                break
        if not found:
            result.append(stack.pop())

    return result[::-1]

Key Points

  • Euler ≠ Hamiltonian: Euler visits every edge once; Hamiltonian visits every vertex once
  • Time: O(E) using Hierholzer's algorithm
  • Check existence first: Verify degree conditions before running the algorithm
  • Applications: Chinese postman problem, DNA sequencing, circuit design
  • Directed version: Even simpler — just follow edges and use pointer array

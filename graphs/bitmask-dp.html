Bitmask DP

Use bitmasks (integers) to represent subsets of elements. dp[mask] stores the answer for the subset encoded by mask. This technique is powerful for problems involving subsets of up to ~20 elements.

Core Concept: Bits as Subsets

For n = 4 elements {0, 1, 2, 3}:

   mask = 1011 (binary) = 11 (decimal)

   Bit 0: 1 → element 0 is IN the subset
   Bit 1: 1 → element 1 is IN the subset
   Bit 2: 0 → element 2 is NOT in the subset
   Bit 3: 1 → element 3 is IN the subset

   So mask = 1011 represents the subset {0, 1, 3}

   Total subsets of n elements = 2^n
   All bits set: (1 << n) - 1 = 1111 = {0,1,2,3}
   Empty set:   0              = 0000 = {}

Problem: Travelling Salesman (TSP)

Given n cities and cost[i][j] (cost to travel from city i to j), find the minimum cost to visit all cities exactly once starting from city 0 and returning to city 0.

Example: 4 cities, cost matrix:
          To:  0    1    2    3
From 0:  [ 0,  10,  15,  20 ]
From 1:  [ 10,  0,  35,  25 ]
From 2:  [ 15,  35,  0,  30 ]
From 3:  [ 20,  25,  30,  0 ]

Brute force: try all (n-1)! = 6 permutations
Bitmask DP:  2^n * n = 64 states (much better for large n)

State & Transition

State: dp[mask][i] = minimum cost to visit exactly the cities in mask, ending at city i

Transition:
For each city j NOT in mask (bit j is 0):
dp[mask | (1<<j)][j] = min(dp[mask | (1<<j)][j], dp[mask][i] + cost[i][j])

Base Case: dp[1][0] = 0 (start at city 0, only city 0 visited)

Answer: min over all i of dp[(1<<n)-1][i] + cost[i][0] (return to start)

Step-by-Step Walkthrough

Step 1: Initialize dp[0001][0] = 0 (at city 0, visited = {0})
Step 2: From mask=0001, city 0: try going to cities 1, 2, 3
• dp[0011][1] = dp[0001][0] + cost[0][1] = 0 + 10 = 10
• dp[0101][2] = dp[0001][0] + cost[0][2] = 0 + 15 = 15
• dp[1001][3] = dp[0001][0] + cost[0][3] = 0 + 20 = 20
Step 3: From mask=0011, city 1: try going to cities 2, 3
• dp[0111][2] = dp[0011][1] + cost[1][2] = 10 + 35 = 45
• dp[1011][3] = dp[0011][1] + cost[1][3] = 10 + 25 = 35
Step 4: Continue expanding until mask = 1111 (all visited). Then add return cost to city 0.
Final answer = min(dp[1111][i] + cost[i][0]) for all i.

Code: TSP with Bitmask DP

// TSP using Bitmask DP
// Time: O(2^n * n^2), Space: O(2^n * n)
int tsp(vector<vector<int>>& cost) {
    int n = cost.size();
    int full = (1 << n) - 1;
    vector<vector<int>> dp(1 << n, vector<int>(n, INT_MAX));

    dp[1][0] = 0; // start at city 0

    for (int mask = 1; mask <= full; mask++) {
        for (int i = 0; i < n; i++) {
            if (dp[mask][i] == INT_MAX) continue;
            if (!(mask & (1 << i))) continue; // i must be in mask

            // Try going to city j not yet visited
            for (int j = 0; j < n; j++) {
                if (mask & (1 << j)) continue; // already visited
                int newMask = mask | (1 << j);
                dp[newMask][j] = min(dp[newMask][j],
                                     dp[mask][i] + cost[i][j]);
            }
        }
    }

    // Return to city 0
    int ans = INT_MAX;
    for (int i = 1; i < n; i++) {
        if (dp[full][i] != INT_MAX)
            ans = min(ans, dp[full][i] + cost[i][0]);
    }
    return ans;
}

// TSP using Bitmask DP
// Time: O(2^n * n^2), Space: O(2^n * n)
static int tsp(int[][] cost) {
    int n = cost.length;
    int full = (1 << n) - 1;
    int[][] dp = new int[1 << n][n];
    for (int[] row : dp) Arrays.fill(row, Integer.MAX_VALUE);

    dp[1][0] = 0; // start at city 0

    for (int mask = 1; mask <= full; mask++) {
        for (int i = 0; i < n; i++) {
            if (dp[mask][i] == Integer.MAX_VALUE) continue;
            if ((mask & (1 << i)) == 0) continue;

            for (int j = 0; j < n; j++) {
                if ((mask & (1 << j)) != 0) continue;
                int newMask = mask | (1 << j);
                dp[newMask][j] = Math.min(dp[newMask][j],
                                          dp[mask][i] + cost[i][j]);
            }
        }
    }

    int ans = Integer.MAX_VALUE;
    for (int i = 1; i < n; i++) {
        if (dp[full][i] != Integer.MAX_VALUE)
            ans = Math.min(ans, dp[full][i] + cost[i][0]);
    }
    return ans;
}

# TSP using Bitmask DP
# Time: O(2^n * n^2), Space: O(2^n * n)
def tsp(cost):
    n = len(cost)
    full = (1 << n) - 1
    dp = [[float('inf')] * n for _ in range(1 << n)]

    dp[1][0] = 0  # start at city 0

    for mask in range(1, full + 1):
        for i in range(n):
            if dp[mask][i] == float('inf'):
                continue
            if not (mask & (1 << i)):
                continue

            # Try going to unvisited city j
            for j in range(n):
                if mask & (1 << j):
                    continue
                new_mask = mask | (1 << j)
                dp[new_mask][j] = min(dp[new_mask][j],
                                      dp[mask][i] + cost[i][j])

    # Return to city 0
    ans = min(dp[full][i] + cost[i][0] for i in range(1, n))
    return ans

Bit Manipulation Tricks

Essential Operations:

Check if bit i is set:     mask & (1 << i)
Set bit i:                 mask | (1 << i)
Clear bit i:               mask & ~(1 << i)
Toggle bit i:              mask ^ (1 << i)
Count set bits:            __builtin_popcount(mask)  // C++
                           Integer.bitCount(mask)    // Java
                           bin(mask).count('1')      // Python

Check if mask is power of 2:  mask & (mask - 1) == 0
Get lowest set bit:           mask & (-mask)
Remove lowest set bit:        mask & (mask - 1)

Iterating Over All Subsets of a Mask

// Iterate all subsets of mask (including empty set)
for (int sub = mask; sub > 0; sub = (sub - 1) & mask) {

    // process sub...

    // process mask...

    int r = mask + c;

Complexity

  • Time: O(2^n * n^2) for TSP. In general O(2^n * n * [transition cost])
  • Space: O(2^n * n)
  • Limitation: Practical only for n ≤ 20 (2^20 = 1M states)
  • Subset iteration: Iterating all subsets of all masks is O(3^n) total

Template: General Bitmask DP

// General bitmask DP template
int n = ...; // number of elements (n ≤ 20)
vector<int> dp(1 << n, /* initial value */);

// Base case
dp[0] = 0; // or dp[initial_mask] = initial_value

for (int mask = 0; mask < (1 << n); mask++) {
    int bits = __builtin_popcount(mask);
    // bits = number of elements chosen so far

    for (int i = 0; i < n; i++) {
        if (mask & (1 << i)) continue; // already chosen
        int newMask = mask | (1 << i);
        dp[newMask] = best(dp[newMask], dp[mask] + cost(bits, i));
    }
}

return dp[(1 << n) - 1]; // all elements chosen

Key Insights

  • When to use: Problems involving subsets of n ≤ 20 elements where order within subset matters or optimal subset selection is needed
  • Processing order: Iterate masks from 0 to 2^n - 1 (smaller subsets before larger ensures dependencies are met)
  • Memory trick: If dp only depends on popcount, can sometimes reduce to dp[mask] instead of dp[mask][extra_dim]
  • Profile-based DP: Bitmask can also encode row/column states in grid problems (e.g., broken profile DP)

Bipartite Graph & 2-Coloring

A bipartite graph is a graph whose vertices can be divided into two disjoint sets (say, Red and Blue) such that every edge connects a vertex in one set to a vertex in the other set. No edge exists between vertices of the same set.

Equivalent definitions (all mean the same thing):
  • Graph is 2-colorable (can color vertices with 2 colors so no adjacent vertices share a color)
  • Graph contains NO odd-length cycle
  • Graph vertices can be split into two independent sets

Visual: Bipartite vs Non-Bipartite

    BIPARTITE (2-colorable):

    Set A (Red)      Set B (Blue)        As a flat graph:

      [0] ------------ [1]                 0(R) --- 1(B)
       |                |                  |         |
       |                |                  |         |
      [2] ------------ [3]                 2(R) --- 3(B)
       |                                   |
       |                                   |
      [4] ------------ [5]                 4(R) --- 5(B)

    Every edge goes Red <-> Blue. No Red-Red or Blue-Blue edges!


    NOT BIPARTITE (has odd cycle):

      0(R) --- 1(B)                Triangle = odd cycle (length 3)
       \       /
        \     /                    Try coloring:
         \   /                       0 = Red
          \ /                        1 = Blue (neighbor of 0)
           2(?)                      2 = ??? Must be Blue (neighbor of 0)
                                            BUT also Red (neighbor of 1)
    Color conflict at vertex 2!              CONTRADICTION!

    Odd cycle = IMPOSSIBLE to 2-color = NOT bipartite

The Key Insight: Odd Cycles

    WHY odd cycles break bipartiteness:

    Even cycle (length 4):          Odd cycle (length 3):

    R --- B                         R --- B
    |     |    Alternating           \   /
    |     |    colors work!           \ /
    B --- R                            ?  <- Must be BOTH colors!

    Even cycle (length 6):          Odd cycle (length 5):

    R - B - R                       R - B - R
    |       |   Works!              |       |
    B - R - B                       B - R --+  <- R connects to R!

    RULE: A graph is bipartite if and only if it contains NO odd-length cycle.

Algorithm: BFS 2-Coloring

To check if a graph is bipartite, try to 2-color it using BFS. If you find a conflict (neighbor already has same color), it is NOT bipartite.

Steps:
  1. Pick any uncolored vertex, color it RED (0), add to BFS queue
  2. For each vertex in queue, color all uncolored neighbors with the OPPOSITE color
  3. If a neighbor is ALREADY colored with the SAME color as current vertex = NOT bipartite!
  4. Repeat for all connected components (graph might be disconnected)
    BFS Coloring Walkthrough:

    Graph: 0-1, 0-2, 1-3, 2-3, 3-4

        0 --- 1
        |     |
        2 --- 3 --- 4

    Step 1: Color 0 = RED, queue = [0]

    Step 2: Process 0 (RED)
      Neighbor 1: uncolored -> BLUE,  queue = [1, 2]
      Neighbor 2: uncolored -> BLUE,  queue = [1, 2]

    Step 3: Process 1 (BLUE)
      Neighbor 0: already RED (opposite of BLUE) -> OK
      Neighbor 3: uncolored -> RED,  queue = [2, 3]

    Step 4: Process 2 (BLUE)
      Neighbor 0: already RED -> OK
      Neighbor 3: already RED (opposite of BLUE) -> OK!

    Step 5: Process 3 (RED)
      Neighbor 1: already BLUE -> OK
      Neighbor 2: already BLUE -> OK
      Neighbor 4: uncolored -> BLUE

    Result: Successfully 2-colored! Graph IS bipartite.
    Set A (RED): {0, 3}    Set B (BLUE): {1, 2, 4}

Implementation

import java.util.*;

public class BipartiteCheck {
    private int V;
    private List<List<Integer>> adj;

    public BipartiteCheck(int v) {
        V = v;
        adj = new ArrayList<>();
        for (int i = 0; i < V; i++)
            adj.add(new ArrayList<>());
    }

    public void addEdge(int u, int v) {
        adj.get(u).add(v);
        adj.get(v).add(u);
    }

    public boolean isBipartite() {
        int[] color = new int[V];
        Arrays.fill(color, -1); // -1 = uncolored, 0 = RED, 1 = BLUE

        // Check all components (graph might be disconnected)
        for (int i = 0; i < V; i++) {
            if (color[i] == -1) {
                if (!bfsCheck(i, color)) {
                    return false;
                }
            }
        }
        return true;
    }

    private boolean bfsCheck(int start, int[] color) {
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(start);
        color[start] = 0; // Color source RED

        while (!queue.isEmpty()) {
            int u = queue.poll();

            for (int v : adj.get(u)) {
                if (color[v] == -1) {
                    // Uncolored: assign opposite color
                    color[v] = 1 - color[u];
                    queue.offer(v);
                } else if (color[v] == color[u]) {
                    // CONFLICT! Same color as neighbor
                    return false; // Not bipartite
                }
            }
        }
        return true;
    }

    // DFS version (alternative)
    public boolean isBipartiteDFS() {
        int[] color = new int[V];
        Arrays.fill(color, -1);

        for (int i = 0; i < V; i++) {
            if (color[i] == -1) {
                color[i] = 0;
                if (!dfsCheck(i, color)) return false;
            }
        }
        return true;
    }

    private boolean dfsCheck(int u, int[] color) {
        for (int v : adj.get(u)) {
            if (color[v] == -1) {
                color[v] = 1 - color[u];
                if (!dfsCheck(v, color)) return false;
            } else if (color[v] == color[u]) {
                return false;
            }
        }
        return true;
    }

    public static void main(String[] args) {
        // Bipartite graph
        BipartiteCheck g1 = new BipartiteCheck(4);
        g1.addEdge(0, 1);
        g1.addEdge(1, 2);
        g1.addEdge(2, 3);
        g1.addEdge(3, 0);
        System.out.println(g1.isBipartite()); // true (even cycle)

        // Non-bipartite graph (triangle)
        BipartiteCheck g2 = new BipartiteCheck(3);
        g2.addEdge(0, 1);
        g2.addEdge(1, 2);
        g2.addEdge(2, 0);
        System.out.println(g2.isBipartite()); // false (odd cycle)
    }
}
                
#include <bits/stdc++.h>
using namespace std;

class BipartiteCheck {
    int V;
    vector<vector<int>> adj;

    bool bfsCheck(int start, vector<int>& color) {
        queue<int> q;
        q.push(start);
        color[start] = 0; // Color source RED

        while (!q.empty()) {
            int u = q.front();
            q.pop();

            for (int v : adj[u]) {
                if (color[v] == -1) {
                    // Uncolored: assign opposite color
                    color[v] = 1 - color[u];
                    q.push(v);
                } else if (color[v] == color[u]) {
                    // CONFLICT! Same color as neighbor
                    return false;
                }
            }
        }
        return true;
    }

public:
    BipartiteCheck(int v) : V(v), adj(v) {}

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    bool isBipartite() {
        vector<int> color(V, -1); // -1 = uncolored

        // Check all connected components
        for (int i = 0; i < V; i++) {
            if (color[i] == -1) {
                if (!bfsCheck(i, color)) {
                    return false;
                }
            }
        }
        return true;
    }

    // DFS version (alternative)
    bool dfsCheck(int u, vector<int>& color) {
        for (int v : adj[u]) {
            if (color[v] == -1) {
                color[v] = 1 - color[u];
                if (!dfsCheck(v, color)) return false;
            } else if (color[v] == color[u]) {
                return false;
            }
        }
        return true;
    }

    bool isBipartiteDFS() {
        vector<int> color(V, -1);
        for (int i = 0; i < V; i++) {
            if (color[i] == -1) {
                color[i] = 0;
                if (!dfsCheck(i, color)) return false;
            }
        }
        return true;
    }
};

int main() {
    // Bipartite graph (even cycle)
    BipartiteCheck g1(4);
    g1.addEdge(0, 1);
    g1.addEdge(1, 2);
    g1.addEdge(2, 3);
    g1.addEdge(3, 0);
    cout << (g1.isBipartite() ? "Bipartite" : "Not Bipartite") << endl;
    // Output: Bipartite

    BipartiteCheck g2(3);
    g2.addEdge(0, 1);
    g2.addEdge(1, 2);
    g2.addEdge(2, 0);
    cout << (g2.isBipartite() ? "Bipartite" : "Not Bipartite") << endl;
    // Output: Not Bipartite

    return 0;
}
                
from collections import deque

def is_bipartite(adj, V):
    color = [-1] * V  # -1 = uncolored

    for start in range(V):
        if color[start] != -1:
            continue
        # BFS coloring
        queue = deque([start])
        color[start] = 0

        while queue:
            node = queue.popleft()
            for neighbor in adj[node]:
                if color[neighbor] == -1:
                    color[neighbor] = 1 - color[node]  # alternate color
                    queue.append(neighbor)
                elif color[neighbor] == color[node]:
                    return False  # same color = odd cycle = NOT bipartite

    return True

# Example
V = 4
adj = [[] for _ in range(V)]
adj[0] = [1, 3]; adj[1] = [0, 2]
adj[2] = [1, 3]; adj[3] = [0, 2]
print("Is bipartite:", is_bipartite(adj, V))  # True

Common Examples of Bipartite Graphs

    Trees are ALWAYS bipartite:

          0(R)                  Level 0: RED
         / \                    Level 1: BLUE
       1(B)  2(B)               Level 2: RED
       /      \                 ...alternating by level!
     3(R)     4(R)


    Even cycles are bipartite:       Odd cycles are NOT:

    R --- B --- R                    R --- B
    |           |                    |     |
    B --- R --- B                    B --- R --- ?
    (length 6: works)                (length 5: fails)


    Complete bipartite graph K(3,3):

    Set A: {a1, a2, a3}    Every vertex in A connects
    Set B: {b1, b2, b3}    to every vertex in B

    a1 --- b1
    a1 --- b2         9 edges total
    a1 --- b3         (3 x 3 = 9)
    a2 --- b1
    ...etc

Complexity

  • Time: O(V + E) — standard BFS/DFS traversal
  • Space: O(V) for the color array and queue/stack

Applications

  • Matching problems: Maximum bipartite matching (Hungarian algorithm) only works on bipartite graphs
  • Job assignment: Workers (set A) matched to jobs (set B)
  • Scheduling: Can tasks be divided into 2 groups with no conflicts within a group?
  • Graph coloring: 2-coloring is the simplest case; if not 2-colorable, need more colors
  • Stable marriage problem: Two disjoint sets with preferences
Interview Pattern: Many problems reduce to "can we split elements into 2 groups such that no conflicts exist within a group?" This is exactly bipartite checking! Examples: "Is the graph 2-colorable?", "Can students be divided into 2 teams with no enemies on same team?"

Quick Reference

Always Bipartite:
  • Trees (any tree is bipartite)
  • Even-length cycles
  • Graphs with no cycles (forests)
  • Paths
Never Bipartite:
  • Odd-length cycles (triangle, pentagon, etc.)
  • Complete graphs K_n for n >= 3
  • Any graph containing an odd cycle as a subgraph

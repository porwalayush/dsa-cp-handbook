Kahn's Algorithm (BFS Topological Sort)

BFS-based topological sort using in-degree (number of incoming edges). Also detects cycles!

The Idea

  In-degree = number of arrows pointing INTO a node

  Graph: 5→0, 5→2, 4→0, 4→1, 2→3, 3→1

  In-degrees:
  Node 0: 2 (from 5, 4)
  Node 1: 2 (from 4, 3)
  Node 2: 1 (from 5)
  Node 3: 1 (from 2)
  Node 4: 0  ← START HERE (no dependencies!)
  Node 5: 0  ← START HERE (no dependencies!)

  Nodes with in-degree 0 = "ready to process" (no prerequisites)

Algorithm Steps

  1. Calculate in-degree of all nodes
  2. Add all nodes with in-degree 0 to queue
  3. While queue not empty:
    • Remove node from queue → add to result
    • For each neighbor: decrease their in-degree by 1
    • If neighbor's in-degree becomes 0 → add to queue
  4. If result size ≠ V → cycle exists!

Visual Walkthrough

Initial: indeg = [2,2,1,1,0,0]. Queue = [4,5]
Process 4: Reduce indeg of 0,1. indeg = [1,1,1,1,-,-]. Queue = [5]
Process 5: Reduce indeg of 0,2. indeg = [0,1,0,1,-,-]. Queue = [0,2] (became 0!)
Process 0: No outgoing edges. Queue = [2]
Process 2: Reduce indeg of 3. indeg = [-,1,-,0,-,-]. Queue = [3]
Process 3: Reduce indeg of 1. indeg = [-,0,-,-,-,-]. Queue = [1]
Process 1: Done!
Result: [4, 5, 0, 2, 3, 1] ✓ 
import java.util.*;

public class KahnsAlgorithm {
    static List<Integer> topoSort(List<List<Integer>> adj, int V) {
        // Step 1: Calculate in-degree of each node
        int[] indegree = new int[V];
        for (int i = 0; i < V; i++)
            for (int nb : adj.get(i))
                indegree[nb]++;

        // Step 2: Add all nodes with in-degree 0 to queue
        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < V; i++)
            if (indegree[i] == 0) queue.add(i);

        List<Integer> result = new ArrayList<>();

        // Step 3: Process
        while (!queue.isEmpty()) {
            int node = queue.poll();
            result.add(node);

            for (int neighbor : adj.get(node)) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0)
                    queue.add(neighbor);
            }
        }

        // Step 4: Check for cycle
        if (result.size() != V) {
            System.out.println("Cycle detected! Topo sort impossible.");
            return new ArrayList<>();
        }

        return result;
    }

    public static void main(String[] args) {
        int V = 6;
        List<List<Integer>> adj = new ArrayList<>();
        for (int i = 0; i < V; i++) adj.add(new ArrayList<>());

        adj.get(5).add(0); adj.get(5).add(2);
        adj.get(4).add(0); adj.get(4).add(1);
        adj.get(2).add(3); adj.get(3).add(1);

        System.out.println("Kahn's Topo Sort: " + topoSort(adj, V));
    }
}

#include <bits/stdc++.h>
using namespace std;

vector<int> topoSort(vector<vector<int>>& adj, int V) {
    vector<int> indegree(V, 0);
    for (int i = 0; i < V; i++)
        for (int nb : adj[i])
            indegree[nb]++;

    queue<int> q;
    for (int i = 0; i < V; i++)
        if (indegree[i] == 0) q.push(i);

    vector<int> result;
    while (!q.empty()) {
        int node = q.front(); q.pop();
        result.push_back(node);

        for (int nb : adj[node]) {
            if (--indegree[nb] == 0)
                q.push(nb);
        }
    }

    if (result.size() != V) {
        cout << "Cycle detected!" << endl;
        return {};
    }
    return result;
}

from collections import deque

def topo_sort(adj, V):
    # Step 1: Calculate in-degree
    indegree = [0] * V
    for i in range(V):
        for nb in adj[i]:
            indegree[nb] += 1

    # Step 2: Add nodes with in-degree 0
    queue = deque(i for i in range(V) if indegree[i] == 0)
    result = []

    # Step 3: Process
    while queue:
        node = queue.popleft()
        result.append(node)
        for neighbor in adj[node]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                queue.append(neighbor)

    # Step 4: Check for cycle
    if len(result) != V:
        print("Cycle detected!")
        return []
    return result

Kahn's vs DFS Topological Sort

  • Kahn's (BFS): Iterative, easy to detect cycles, gives level-wise ordering
  • DFS: Recursive, simpler code, but cycle detection needs extra coloring
  • Both: O(V + E) time, O(V) space
  • Use Kahn's when: You need to detect cycle + topo sort simultaneously, or need level-by-level processing
